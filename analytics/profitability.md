# Рентабельность

Раздел «Рентабельность» предоставляет инструменты для анализа финансовой эффективности сервисного центра. Здесь можно оценить прибыльность отдельных ремонтов, маржинальность услуг и результативность работы сотрудников.

## Обзор раздела

Страница рентабельности объединяет несколько аналитических блоков, каждый из которых раскрывает определённый аспект финансовых показателей. Данные обновляются в реальном времени и учитывают выбранный период.

<!-- screenshot: общий вид раздела рентабельности -->

## Рентабельность ремонтов

Анализ рентабельности на уровне отдельных ремонтов показывает, сколько прибыли приносит каждый заказ. Система рассчитывает разницу между стоимостью ремонта для клиента и себестоимостью выполненных работ и использованных запчастей.

{% note info %}

Для корректного расчёта рентабельности необходимо, чтобы в системе были указаны закупочные цены запчастей и себестоимость работ. Эти данные задаются в разделах «Склад» и «Настройки».

{% endnote %}

## Маржинальность услуг

Блок маржинальности услуг показывает, какие виды работ приносят наибольшую прибыль. Для каждой услуги отображается средняя маржа в процентах и абсолютном выражении, что помогает определить наиболее выгодные направления.

<!-- screenshot: таблица маржинальности услуг с процентами и суммами -->

## Анализ наценки на запчасти

Система анализирует наценку на запчасти, сравнивая закупочную стоимость с ценой продажи клиенту. Отчёт позволяет выявить позиции с низкой или чрезмерно высокой наценкой и скорректировать ценовую политику.

{% note info %}

Наценка рассчитывается автоматически на основании закупочной цены из складского учёта и цены, указанной в ремонтном заказе. Если закупочная цена не заполнена, позиция исключается из анализа.

{% endnote %}

<!-- screenshot: отчёт по наценке на запчасти с указанием закупочной и продажной цен -->

## Эффективность сотрудников

Раздел эффективности сотрудников позволяет оценить вклад каждого мастера в финансовые результаты. Для каждого сотрудника отображается количество выполненных ремонтов, суммарная выручка и средняя рентабельность.

<!-- screenshot: таблица эффективности сотрудников с показателями выручки и рентабельности -->

## Рейтинг мастеров

На основании данных об эффективности формируется рейтинг мастеров. Рейтинг учитывает не только объём выполненных работ, но и качество — процент возвратов и повторных ремонтов.

{% note info %}

Рейтинг мастеров обновляется ежедневно. Данные доступны руководителям и администраторам в зависимости от настроек ролей доступа.

{% endnote %}

<!-- screenshot: рейтинг мастеров с показателями объёма и качества -->

## Сравнение периодов

Инструмент сравнения периодов позволяет сопоставить финансовые показатели за два произвольных диапазона дат. Это помогает оценить динамику рентабельности, выявить сезонные тренды и измерить эффект от принятых управленческих решений.

<!-- screenshot: сравнение показателей рентабельности за два периода -->

## Детализация по категориям

Рентабельность можно анализировать в разрезе категорий устройств и типов ремонтов. Детализация помогает понять, какие направления работы наиболее прибыльны, а какие требуют пересмотра ценовой политики.

<!-- screenshot: детализация рентабельности по категориям устройств -->

## Экспорт отчётов

Все аналитические данные по рентабельности доступны для экспорта. Отчёты можно выгрузить в табличном формате для дальнейшей обработки или предоставления руководству.
