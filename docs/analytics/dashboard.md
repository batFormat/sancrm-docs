# Дашборд

Дашборд — это главная аналитическая панель SanCRM, которая отображает ключевые показатели работы сервисного центра. Здесь собрана сводная информация, позволяющая быстро оценить текущее состояние бизнеса.

## Обзор дашборда

Дашборд доступен из главного меню системы и содержит набор виджетов. Каждый виджет отвечает за определённую группу показателей.

## Виджеты дашборда

### Выручка за сегодня

Виджет отображает суммарную выручку от закрытых ремонтов. Доступны пресеты: сегодня, вчера, 7 дней, 30 дней, 60 дней.

### Выручка по филиалам

Для каждого филиала отображается отдельный виджет выручки. Это позволяет сравнивать показатели между точками обслуживания.

### Ремонты в работе

Диаграмма распределения текущих ремонтов по группам статусов. Показывает, сколько заказов находится на каждом этапе.

### Мало на складе

Количество товаров с остатком ниже минимального порога. Помогает контролировать наличие запчастей.

### Тренд закрытых ремонтов

Линейный график количества закрытых ремонтов по дням. Доступны пресеты: 7, 14, 30 и 60 дней.

### Лента активности

Хронологическая лента переходов ремонтов между статусами. Доступны фильтры: по статусам, по сотруднику, только с указанной причиной. Для каждого перехода отображается номер ремонта, устройство, сотрудник и время.

## Фильтрация по периоду

Виджеты поддерживают выбор периода из набора пресетов: сегодня, вчера, 7 дней, 14 дней, 30 дней, 60 дней.

{% note info %}

Произвольный выбор дат (с календарём) не поддерживается. Используйте доступные пресеты для просмотра данных за нужный период.

{% endnote %}

## Сравнение филиалов

Если в системе настроено несколько филиалов, дашборд автоматически создаёт виджет выручки для каждого филиала. Это позволяет сравнивать показатели на одной странице.

## Отчёты через Telegram

Аналитические данные можно получать в виде регулярных отчётов в Telegram. Система автоматически отправляет отчёты о рентабельности (ежедневные, еженедельные, ежемесячные) в настроенный Telegram-чат.

{% note info %}

Экспорт данных дашборда в файл (Excel, CSV, PDF) не поддерживается. Для получения отчётов используйте Telegram-интеграцию.

{% endnote %}
