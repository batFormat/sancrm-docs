# Дашборд

Дашборд — это главная аналитическая панель SanCRM, которая отображает ключевые показатели работы сервисного центра в реальном времени. Здесь собрана сводная информация, позволяющая быстро оценить текущее состояние бизнеса.

## Обзор дашборда

Дашборд доступен из главного меню системы и открывается на отдельной странице с набором виджетов. Каждый виджет отвечает за определённую группу показателей и обновляется автоматически.

<!-- screenshot: общий вид дашборда с виджетами -->

## Ключевые метрики

На дашборде представлены основные числовые показатели, которые дают моментальный срез по деятельности сервисного центра. К ним относятся: количество ремонтов, выручка за период и среднее время ремонта.

{% note info %}

Все метрики рассчитываются с учётом выбранного периода и филиала. При изменении фильтров значения пересчитываются автоматически.

{% endnote %}

<!-- screenshot: блок ключевых метрик — количество ремонтов, выручка, среднее время -->

## Количество ремонтов

Виджет отображает общее число ремонтов за выбранный период с разбивкой по статусам: новые, в работе, завершённые и выданные. Дополнительно показывается динамика по сравнению с предыдущим аналогичным периодом.

<!-- screenshot: виджет количества ремонтов с разбивкой по статусам -->

## Выручка

Виджет выручки показывает суммарный доход за период, включая стоимость работ и проданных запчастей. График позволяет увидеть динамику выручки по дням или неделям.

<!-- screenshot: виджет выручки с графиком динамики -->

## Среднее время ремонта

Показатель среднего времени ремонта рассчитывается от момента приёма устройства до его выдачи клиенту. Виджет помогает отслеживать скорость обслуживания и выявлять задержки.

{% note info %}

Среднее время ремонта учитывает только завершённые ремонты. Ремонты в статусе «ожидание запчастей» могут быть исключены из расчёта в зависимости от настроек.

{% endnote %}

<!-- screenshot: виджет среднего времени ремонта -->

## Виджеты и графики

Помимо числовых показателей, дашборд содержит графические виджеты: круговые диаграммы распределения ремонтов по категориям, столбчатые графики загрузки мастеров и линейные графики динамики обращений.

<!-- screenshot: графические виджеты — диаграммы и графики -->

## Фильтрация по периоду

Дашборд поддерживает выбор произвольного диапазона дат. Доступны быстрые пресеты: сегодня, вчера, текущая неделя, текущий месяц, а также возможность указать произвольные даты начала и конца периода.

<!-- screenshot: панель выбора диапазона дат с пресетами -->

## Сравнение филиалов

Если в системе настроено несколько филиалов, дашборд позволяет сравнивать их показатели на одной странице. Это помогает выявить различия в загрузке, выручке и скорости обслуживания между точками.

{% note info %}

Для сравнения филиалов необходимо, чтобы в системе было зарегистрировано более одного филиала. Настройка филиалов выполняется в разделе «Настройки».

{% endnote %}

<!-- screenshot: режим сравнения филиалов на дашборде -->

## Экспорт данных

Данные дашборда можно выгрузить для дальнейшего анализа. Система поддерживает экспорт сводных показателей в табличном формате, что позволяет использовать их во внешних отчётах.
