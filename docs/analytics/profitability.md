# Рентабельность

Раздел «Рентабельность» предоставляет информацию о финансовой эффективности сервисного центра. Отчёты формируются автоматически и отправляются в Telegram.

## Отчёт о рентабельности

Система автоматически рассчитывает показатели рентабельности и отправляет отчёт в настроенный Telegram-чат. Отчёт включает:

- **Выручка от ремонтов** — суммарный доход от завершённых ремонтов.
- **Выручка от проката** — суммарный доход от проката инструмента.
- **Переменные расходы** — себестоимость запчастей и материалов.
- **Постоянные расходы** — фиксированные затраты (при наличии данных).
- **Прибыль** — разница между выручкой и расходами.
- **Маржинальность** — процент прибыли от выручки.

## Периодичность отчётов

Система поддерживает автоматическую отправку отчётов с разной периодичностью.

- **Ежедневный** — отчёт за прошедший день.
- **Еженедельный** — отчёт за прошедшую неделю.
- **Ежемесячный** — отчёт за прошедший месяц.

{% note info %}

Для получения отчётов необходимо настроить Telegram-интеграцию и указать ID чата для отправки. Подробнее — в разделе [Интеграции](../integrations/).

{% endnote %}

{% note info %}

Экспорт отчётов в файл (Excel, CSV, PDF) не поддерживается. Данные о рентабельности доступны только через Telegram-отчёты.

{% endnote %}
