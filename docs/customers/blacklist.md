# Чёрный список

Функция чёрного списка позволяет отмечать проблемных клиентов, с которыми нежелательно вести работу. Добавление клиента в чёрный список не удаляет его из системы, но информирует сотрудников при последующих обращениях.

## Назначение чёрного списка

Чёрный список предназначен для защиты сервисного центра от недобросовестных клиентов. Это могут быть клиенты, которые не забирают технику, отказываются оплачивать работу или ведут себя неподобающим образом.

{% note warning %}

Добавление клиента в чёрный список **блокирует** создание ремонтов и прокатов для него. При попытке оформить заказ на клиента из чёрного списка система выдаст ошибку и отклонит операцию. Для продолжения работы с клиентом его необходимо сначала убрать из чёрного списка.

{% endnote %}

## Добавление в чёрный список

Клиента можно внести в чёрный список из его карточки. При добавлении необходимо указать причину — это поможет другим сотрудникам понять контекст ситуации.

<!-- screenshot: кнопка добавления в чёрный список в карточке клиента -->

## Указание причины

При добавлении в чёрный список система запрашивает причину блокировки. Причина сохраняется в истории клиента и отображается вместе с предупреждением. Рекомендуется описывать причину максимально конкретно.

<!-- screenshot: модальное окно с полем ввода причины добавления в чёрный список -->

## Предупреждения при создании ремонта

Когда сотрудник пытается создать ремонт или прокат для клиента из чёрного списка, система блокирует операцию и выдаёт сообщение об ошибке с указанием, что клиент находится в чёрном списке.

<!-- screenshot: предупреждение о клиенте из чёрного списка при создании ремонта -->

## Удаление из чёрного списка

Клиента можно убрать из чёрного списка, если обстоятельства изменились. Удаление выполняется из карточки клиента. Информация о нахождении в чёрном списке сохраняется в истории клиента для справки.

<!-- screenshot: кнопка удаления из чёрного списка в карточке клиента -->

## Просмотр чёрного списка

Полный перечень клиентов, находящихся в чёрном списке, доступен через фильтрацию в разделе «Клиенты». Для каждого клиента отображается причина и дата добавления.
