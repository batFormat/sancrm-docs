# Создание клиента

В этом разделе описан процесс добавления нового клиента в систему SanCRM. Клиент может быть создан вручную из раздела «Клиенты» или автоматически при оформлении нового ремонта.

## Способы создания клиента

Существует несколько способов добавить нового клиента в базу. Клиента можно создать из раздела «Клиенты», а также непосредственно при оформлении ремонта или проката — система предложит добавить нового клиента, если по введённому номеру телефона совпадений не найдено.

<!-- screenshot: кнопка «Добавить клиента» в разделе клиентов -->

## Обязательные поля

При создании клиента необходимо заполнить минимальный набор полей: имя клиента и номер телефона. Без этих данных сохранение карточки невозможно.

{% note info %}

Система автоматически проверяет уникальность номера телефона. Если клиент с таким номером уже существует, будет предложено перейти к его карточке вместо создания дубликата.

{% endnote %}

<!-- screenshot: форма создания клиента с обязательными полями -->

## Номер телефона

Телефон является основным идентификатором клиента. При вводе номера система выполняет поиск по существующей базе и показывает совпадения. Поддерживается ввод нескольких номеров телефона для одного клиента.

<!-- screenshot: поле ввода телефона с подсказкой существующего клиента -->

## Адрес электронной почты

Электронная почта используется для отправки уведомлений о статусе ремонта и чеков. Поле не является обязательным, но рекомендуется к заполнению для полноценного информирования клиента.

<!-- screenshot: поле ввода e-mail в форме создания клиента -->

## Дополнительная информация

При создании клиента можно указать дополнительные сведения: адрес, источник привлечения и другие данные. Эти поля помогают вести более детальный учёт клиентской базы.

<!-- screenshot: дополнительные поля в форме создания клиента -->

## Заметки

К новому клиенту можно сразу добавить текстовую заметку. Заметки полезны для фиксации особенностей общения, предпочтений или договорённостей с клиентом.

{% note info %}

Заметки видны всем сотрудникам организации. Не храните в заметках конфиденциальные данные, такие как пароли или данные банковских карт.

{% endnote %}
