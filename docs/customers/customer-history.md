# История клиента

Раздел истории клиента предоставляет информацию о связанных ремонтах и прокатах. Из карточки клиента можно быстро перейти к любому заказу.

## Прошлые ремонты

В карточке клиента перечислены все ремонты, которые были оформлены на данного клиента. Для каждого ремонта указаны: номер заказа, устройство, статус и дата. По клику можно перейти к карточке ремонта.

{% note info %}

Список ремонтов сортируется по дате создания — от новых к старым. Это позволяет быстро найти последнее обращение клиента.

{% endnote %}

## Прокаты

Все операции проката, связанные с клиентом, отображаются отдельным списком. Для каждого проката указаны: инструмент, дата выдачи, дата возврата и текущий статус.
