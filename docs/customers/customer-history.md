# История клиента

Раздел истории клиента предоставляет полную хронологию взаимодействий с клиентом. Здесь собраны все ремонты, покупки, прокаты и другие события, связанные с данным клиентом.

## Хронология взаимодействий

История клиента отображается в виде хронологической ленты событий. Каждое событие содержит дату, тип действия и краткое описание. Лента позволяет быстро оценить частоту обращений и характер взаимодействий.

<!-- screenshot: хронологическая лента событий в карточке клиента -->

## Прошлые ремонты

В истории клиента перечислены все ремонты, которые были оформлены на данного клиента. Для каждого ремонта указаны: номер заказа, устройство, статус и дата. По клику можно перейти к карточке ремонта.

{% note info %}

Список ремонтов сортируется по дате создания — от новых к старым. Это позволяет быстро найти последнее обращение клиента.

{% endnote %}

<!-- screenshot: список прошлых ремонтов клиента с номерами заказов и статусами -->

## Покупки и продажи

Если клиент приобретал товары или запасные части, эти операции также фиксируются в истории. Отображаются наименование товара, количество, сумма и дата продажи.

<!-- screenshot: список покупок клиента -->

## Прокаты

Все операции проката, связанные с клиентом, отображаются в отдельном блоке истории. Для каждого проката указаны: предмет, дата выдачи, дата возврата и текущий статус.

<!-- screenshot: список прокатов клиента -->

## Финансовая сводка

В истории клиента доступна сводка по финансовым показателям: общая сумма оплаченных ремонтов, сумма покупок и задолженности, если таковые имеются. Эти данные помогают оценить ценность клиента для бизнеса.

{% note info %}

Финансовая сводка рассчитывается автоматически на основе завершённых заказов и проведённых оплат. Отменённые и неоплаченные заказы в итоговую сумму не включаются.

{% endnote %}

<!-- screenshot: финансовая сводка клиента — общая сумма, количество заказов -->

## Фильтрация истории

Историю клиента можно фильтровать по типу события: только ремонты, только покупки, только прокаты или все события. Также доступна фильтрация по дате для просмотра активности за определённый период.
