# Обзор модуля «Клиенты»

Модуль «Клиенты» предназначен для ведения клиентской базы сервисного центра. Здесь хранятся контактные данные, история обращений и вся информация, необходимая для качественного обслуживания.

## Список клиентов

В разделе отображается полный перечень клиентов с возможностью сортировки и фильтрации. Каждая строка списка содержит основные сведения: имя, телефон и дату последнего обращения.

<!-- screenshot: список клиентов с колонками имя, телефон, дата последнего обращения -->

## Карточка клиента

Карточка клиента — это центральная страница с полной информацией о клиенте. Здесь собраны контактные данные, заметки, связанные ремонты и история взаимодействий.

{% note info %}

Карточка клиента открывается по клику на имя клиента в любом разделе системы — из списка ремонтов, проката или журнала событий.

{% endnote %}

<!-- screenshot: карточка клиента с основными полями и вкладками -->

## Поиск и фильтрация

Система поддерживает быстрый поиск клиентов по имени, номеру телефона или адресу электронной почты. Также доступны расширенные фильтры для сужения результатов по различным критериям.

<!-- screenshot: строка поиска клиентов и панель фильтров -->

## Контактная информация

В карточке клиента хранятся контактные данные: номер телефона, адрес электронной почты и физический адрес. Телефон является основным идентификатором клиента в системе.

{% note info %}

Номер телефона является основным идентификатором клиента в системе. По нему осуществляется автоматическое определение клиента при создании нового ремонта.

{% endnote %}

<!-- screenshot: раздел контактной информации в карточке клиента -->

## Заметки по клиенту

К каждому клиенту можно добавлять произвольные текстовые заметки. Заметки видны всем сотрудникам и помогают фиксировать важные детали, которые не укладываются в стандартные поля карточки.
