# Частые вопросы

В этом разделе собраны ответы на наиболее популярные вопросы по работе с SanCRM. Если вы не нашли ответ на свой вопрос, обратитесь в [техническую поддержку](#техническая-поддержка).

## Общие вопросы

{% cut "Что такое SanCRM?" %}

SanCRM — это CRM-система, разработанная специально для сервисных центров и мастерских. Она позволяет вести учёт ремонтов, клиентов, склада, проката и аналитики в едином интерфейсе.

{% endcut %}

{% cut "Как зарегистрироваться в системе?" %}

Для регистрации перейдите на страницу входа и нажмите кнопку «Регистрация». Подробная инструкция описана в разделе [Регистрация и вход](../quickstart/registration.md).

{% endcut %}

{% cut "Какие браузеры поддерживаются?" %}

SanCRM поддерживает актуальные версии Google Chrome, Mozilla Firefox, Microsoft Edge и Safari. Рекомендуем использовать последнюю версию Google Chrome для наилучшей производительности.

{% endcut %}

Интерфейс SanCRM адаптирован для работы на планшетах и мобильных устройствах. Основные функции доступны через мобильный браузер.

## Ремонты

{% cut "Как изменить статус ремонта?" %}

Статус ремонта можно изменить на странице ремонта, нажав на кнопку текущего статуса и выбрав новый из списка допустимых переходов. Подробнее — в разделе [Статусы и переходы](../repairs/statuses.md).

{% endcut %}

{% cut "Как распечатать квитанцию?" %}

Откройте карточку ремонта и нажмите кнопку «Печать». Выберите нужный шаблон документа — квитанцию, акт приёма или гарантийный талон. Подробнее — в разделе [Квитанции и документы](../repairs/documents.md).

{% endcut %}

{% cut "Как добавить запчасти к ремонту?" %}

В карточке ремонта перейдите на вкладку «Запчасти» и нажмите «Добавить». Выберите товар со склада или введите данные вручную. Подробнее — в разделе [Запчасти и товары](../repairs/spare-parts.md).

{% endcut %}

{% cut "Можно ли вернуть ремонт в предыдущий статус?" %}

<!-- TODO: Уточнить логику возврата статусов -->

Возможность возврата зависит от настроенных переходов статусов в вашей организации. Администратор может настроить обратные переходы в разделе [Настройки](../settings/).

{% endcut %}

## Склад

{% cut "Как провести инвентаризацию?" %}

Для проведения инвентаризации перейдите в раздел «Склад» и выберите пункт «Инвентаризация». Создайте новый документ инвентаризации и заполните фактические остатки. Подробнее — в разделе [Склад](../warehouse/).

{% endcut %}

{% cut "Как импортировать товары?" %}

SanCRM поддерживает импорт товаров из файлов CSV и Excel. Перейдите в раздел «Склад», нажмите «Импорт» и загрузите файл с данными. Подробнее — в разделе [Склад](../warehouse/).

{% endcut %}

{% cut "Как настроить автоматическое списание запчастей?" %}

<!-- TODO: Описать настройку автоматического списания при смене статуса -->

Автоматическое списание запчастей можно настроить через действия Workflow. При переходе ремонта в определённый статус запчасти будут списаны со склада автоматически.

{% endcut %}

## Оплата и подписка

{% cut "Сколько стоит SanCRM?" %}

<!-- TODO: Добавить актуальную информацию о тарифах и ценах -->

Актуальные тарифы и цены доступны на официальном сайте SanCRM. Стоимость зависит от выбранного плана и количества пользователей.

{% endcut %}

{% cut "Как сменить тарифный план?" %}

Смена тарифного плана доступна в разделе «Настройки» -> «Подписка». Выберите новый план и подтвердите изменение. Новый тариф вступит в силу с начала следующего расчётного периода.

{% endcut %}

{% cut "Есть ли пробный период?" %}

<!-- TODO: Уточнить условия пробного периода -->

Да, SanCRM предоставляет бесплатный пробный период для новых пользователей. В течение пробного периода доступен полный функционал системы без ограничений.

{% endcut %}

{% cut "Какие способы оплаты поддерживаются?" %}

<!-- TODO: Перечислить доступные способы оплаты -->

SanCRM поддерживает оплату банковскими картами и безналичным переводом для юридических лиц. Подробности можно уточнить у службы поддержки.

{% endcut %}

## Техническая поддержка

{% cut "Как связаться с технической поддержкой?" %}

<!-- TODO: Добавить актуальные контакты поддержки -->

Связаться с поддержкой можно через чат в интерфейсе SanCRM, по электронной почте или через Telegram. Контактная информация доступна в разделе «Помощь» в меню системы.

{% endcut %}

{% cut "Каково время ответа поддержки?" %}

<!-- TODO: Уточнить SLA по времени ответа -->

Среднее время ответа технической поддержки зависит от тарифного плана. Обращения обрабатываются в порядке поступления в рабочие часы.

{% endcut %}

{% cut "Как сообщить об ошибке?" %}

Для сообщения об ошибке обратитесь в техническую поддержку, указав описание проблемы, шаги для воспроизведения и скриншоты при наличии. Это поможет оперативно диагностировать и устранить неполадку.
