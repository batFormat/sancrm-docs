# Частые вопросы

В этом разделе собраны ответы на наиболее популярные вопросы по работе с SanCRM.

## Общие вопросы

{% cut "Что такое SanCRM?" %}

SanCRM — это CRM-система, разработанная специально для сервисных центров и мастерских по ремонту инструмента. Она позволяет вести учёт ремонтов, клиентов, склада и проката в едином интерфейсе.

{% endcut %}

{% cut "Как зарегистрироваться в системе?" %}

Для регистрации перейдите на страницу входа и нажмите кнопку «Регистрация». Подробная инструкция описана в разделе [Регистрация и вход](../quickstart/registration.md).

{% endcut %}

{% cut "Какие браузеры поддерживаются?" %}

SanCRM поддерживает актуальные версии Google Chrome, Mozilla Firefox, Microsoft Edge и Safari. Рекомендуем использовать последнюю версию Google Chrome для наилучшей производительности.

{% endcut %}

{% cut "Можно ли работать с мобильного устройства?" %}

Интерфейс SanCRM адаптирован для работы на планшетах. Основные функции доступны через мобильный браузер, однако для полноценной работы рекомендуется использовать компьютер.

{% endcut %}

## Ремонты

{% cut "Как изменить статус ремонта?" %}

Статус ремонта изменяется через кнопки действий (Workflow) на карточке ремонта. Доступные переходы определяются текущим статусом и настроенными действиями. Подробнее — в разделе [Статусы и переходы](../repairs/statuses.md).

{% endcut %}

{% cut "Как распечатать квитанцию?" %}

Откройте карточку ремонта и нажмите кнопку «Печать». Доступны два типа квитанций: полная и краткая. Подробнее — в разделе [Квитанции и документы](../repairs/documents.md).

{% endcut %}

{% cut "Как добавить запчасти к ремонту?" %}

В карточке ремонта перейдите на вкладку «Запчасти» и нажмите «Добавить». Выберите товар со склада — запчасть будет списана автоматически. Подробнее — в разделе [Запчасти и товары](../repairs/spare-parts.md).

{% endcut %}

{% cut "Можно ли вернуть ремонт в предыдущий статус?" %}

Возможность перехода между статусами определяется настроенными действиями Workflow. Администратор может создать действие с обратным переходом. Подробнее — в разделе [Действия Workflow](../settings/workflow.md).

{% endcut %}

## Склад

{% cut "Как провести инвентаризацию?" %}

Для проведения инвентаризации перейдите в раздел «Склад» и создайте документ инвентаризации. Заполните фактические остатки — система рассчитает расхождения. Подробнее — в разделе [Склад](../warehouse/).

{% endcut %}

{% cut "Как импортировать товары?" %}

SanCRM поддерживает импорт товаров из файлов CSV и Excel. Перейдите в раздел «Склад», нажмите «Импорт» и загрузите файл с данными. Подробнее — в разделе [Склад](../warehouse/).

{% endcut %}

{% cut "Как списать запчасти на ремонт?" %}

Запчасти списываются со склада при добавлении к ремонту через вкладку «Запчасти» в карточке ремонта. Списание происходит автоматически из выбранного склада.

{% endcut %}

## Оплата и подписка

{% cut "Сколько стоит SanCRM?" %}

Актуальные тарифы и цены доступны на официальном сайте SanCRM. Стоимость зависит от выбранного плана и количества пользователей.

{% endcut %}

{% cut "Как сменить тарифный план?" %}

Смена тарифного плана доступна в разделе «Настройки» -> «Подписка». Выберите новый план и подтвердите изменение.

{% endcut %}

{% cut "Есть ли пробный период?" %}

Да, SanCRM предоставляет бесплатный пробный период для новых пользователей. В течение пробного периода доступен полный функционал системы.

{% endcut %}

## Техническая поддержка

{% cut "Как связаться с технической поддержкой?" %}

Связаться с поддержкой можно через чат в интерфейсе SanCRM или через Telegram. Контактная информация доступна в разделе «Помощь» в меню системы.

{% endcut %}

{% cut "Как сообщить об ошибке?" %}

Для сообщения об ошибке обратитесь в техническую поддержку, указав описание проблемы, шаги для воспроизведения и скриншоты при наличии. Это поможет оперативно диагностировать и устранить неполадку.

{% endcut %}
