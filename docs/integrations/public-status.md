# Публичная проверка статуса

SanCRM предоставляет публичную страницу, на которой клиенты могут самостоятельно проверить текущий статус своего ремонта. Страница не требует авторизации и доступна по прямой ссылке.

## Как это работает

Клиент переходит по ссылке, выбирает префикс филиала из выпадающего списка, вводит номер заказа и получает актуальную информацию о ходе ремонта.

<!-- screenshot: публичная страница проверки статуса — поле ввода номера заказа -->

## Адрес страницы

Страница доступна по адресу:

```
https://<ваш-домен>/repair-status
```

Ссылку можно разместить на сайте сервисного центра, в подписи электронной почты или в мессенджерах.

{% note info %}

Страница проверки статуса доступна всегда — отдельного переключателя для её включения или отключения нет.

{% endnote %}

## Отображаемая информация

При вводе номера заказа клиенту показывается ограниченный набор данных:

- **Номер ремонта** — форматированный код (например, «C-11508»).
- **Категория** — тип устройства.
- **Бренд** — производитель.
- **Статус** — текущий статус ремонта в виде метки.

Если ремонт находится в финальном статусе, вместо подробной информации отображается сообщение о завершении.

{% note info %}

Внутренние комментарии, финансовые данные и контактная информация клиента не отображаются на публичной странице.

{% endnote %}

## Оформление

На странице отображается логотип организации, если он загружен в настройках. Цветовая схема страницы фиксированная и не настраивается.
