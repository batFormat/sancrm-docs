# Telegram-уведомления

SanCRM отправляет служебные уведомления в Telegram-чат сотрудников. Интеграция работает в одностороннем режиме — система отправляет сообщения, но не принимает команды и не взаимодействует с клиентами.

## Настройка подключения

Для работы интеграции необходимо указать два параметра в конфигурации:

- **Токен бота** — получается при создании бота через BotFather в Telegram.
- **ID чата** — идентификатор группового чата, в который бот будет отправлять уведомления.

Бот необходимо добавить в групповой чат и предоставить ему права на отправку сообщений.

{% note info %}

Один бот и один чат настраиваются на уровне всей системы. Все уведомления поступают в один общий чат.

{% endnote %}

## Типы уведомлений

Система отправляет три типа сообщений в Telegram.

### Уведомления о незвонках

Система проверяет ремонты в статусах «Готов» и «Подбор», у которых отмечен флаг необходимости обратного звонка. По каждому такому ремонту отправляется сообщение с информацией:

- Номер и код ремонта.
- Категория, бренд и модель устройства.
- Имя и телефон клиента.
- Кнопка «Открыть в CRM» — прямая ссылка на карточку ремонта.

Проверка выполняется автоматически по расписанию в рабочие дни.

### Отчёты о рентабельности

Финансовые отчёты отправляются автоматически по расписанию:

| Отчёт | Расписание |
|---|---|
| Ежедневный (по филиалу) | Каждый день в 09:00, кроме понедельника |
| Еженедельный (по филиалу) | Понедельник в 09:30 |
| Ежемесячный (по филиалу) | 1-е число месяца в 09:30 |

Каждый отчёт содержит:

- Название филиала и период.
- Выручка: ремонты и прокат.
- Расходы: переменные (запчасти) и постоянные.
- Прибыль и маржинальность.
- Количество ремонтов и прокатов.

После отчётов по филиалам отправляется сводка по всей сети.

### Уведомления о резервном копировании

При сбое резервного копирования базы данных в Telegram поступает оповещение. Для этих уведомлений можно настроить отдельный чат.

## Ограничения

- Telegram используется только для уведомлений сотрудников. Клиенты не получают сообщений через Telegram.
- Бот не принимает входящие сообщения и не поддерживает команды.
- Привязка клиентов к Telegram не реализована.
- SMS и email-уведомления не поддерживаются.

{% note info %}

Для информирования клиентов о статусе ремонта используйте [публичную страницу проверки статуса](public-status.md).

{% endnote %}
