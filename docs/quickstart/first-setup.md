# Первоначальная настройка

После регистрации необходимо выполнить базовую настройку системы. В этом разделе описаны ключевые шаги: создание организации, добавление филиалов, настройка ролей сотрудников и конфигурация статусов ремонтов.

## Создание организации

Первый шаг после входа — создание профиля вашей организации. Здесь будет описано, как указать название компании, контактные данные, юридическую информацию и загрузить логотип, который будет отображаться в документах и квитанциях.

{% note info %}

Данные организации используются при печати квитанций, поэтому заполняйте их внимательно.

{% endnote %}

## Настройка ролей сотрудников

SanCRM использует три предустановленные роли для разграничения доступа: администратор, мастер и приёмщик. Здесь будет описано, как создавать учётные записи сотрудников и назначать им роли.

{% note info %}

Роли фиксированные — создание собственных ролей не поддерживается. Каждая роль определяет набор доступных разделов и действий.

{% endnote %}

## Настройка статусов ремонтов

Статусы ремонтов определяют этапы прохождения заказа через вашу мастерскую. В этом разделе будет описано, как просмотреть стандартные статусы, создать новые, настроить их порядок и цветовое обозначение для удобной визуализации на доске ремонтов.
