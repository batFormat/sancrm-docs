# Регистрация и вход

В этом разделе описан процесс создания учётной записи в SanCRM, подтверждения электронной почты и входа в систему. Следуя этим шагам, вы получите доступ ко всем возможностям платформы.

## Создание учётной записи

Для начала работы с SanCRM необходимо зарегистрировать новую учётную запись. Здесь будет описан пошаговый процесс заполнения регистрационной формы: указание имени, электронной почты и пароля.

{% note info %}

Рекомендуется использовать корпоративную электронную почту для регистрации — это упростит совместную работу с коллегами.

{% endnote %}
