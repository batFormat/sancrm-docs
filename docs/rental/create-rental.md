# Оформление проката

Оформление проката — основная операция модуля, в ходе которой фиксируется выдача инструмента клиенту. Форма заказа собирает информацию о клиенте, выбранных позициях, сроках и условиях аренды.

<!-- screenshot: кнопка создания нового проката на панели инструментов -->

## Открытие формы оформления

Новый заказ на прокат можно создать несколькими способами из разных частей интерфейса.

- Кнопка «Новый прокат» на главной панели модуля.
- Быстрое создание из карточки клиента с автоматической привязкой.
- Создание из карточки инструмента, если клиент выбирает конкретную позицию.

<!-- screenshot: форма оформления нового проката -->

## Выбор клиента

При оформлении проката необходимо указать клиента, которому выдаётся оборудование. Система поддерживает поиск по существующей базе и создание нового клиента.

- **Поиск существующего клиента** — по имени, телефону или номеру документа с автоподсказками.
- **Создание нового клиента** — быстрая форма без необходимости покидать страницу оформления.
- **Документ, удостоверяющий личность** — обязательное поле для фиксации паспортных данных.

{% note info %}

Если клиент обращается повторно, система покажет историю его предыдущих заказов и информацию о просроченных возвратах. Это помогает оценить надёжность клиента при принятии решения о выдаче.

{% endnote %}

## Выбор позиций

Блок выбора инструмента позволяет добавить одну или несколько позиций из каталога в заказ.

- **Поиск по каталогу** — быстрый поиск по названию, категории или инвентарному номеру.
- **Фильтрация по доступности** — отображаются только позиции в статусе «Доступен».
- **Множественный выбор** — возможность добавить несколько единиц оборудования в один заказ.
- **Проверка комплектации** — при выборе позиции отображается её комплектация для сверки при выдаче.

<!-- screenshot: выбор инструмента из каталога с фильтрами -->

{% note warning %}

Перед выдачей инструмента обязательно проверьте комплектацию и зафиксируйте текущее состояние оборудования. Это защитит вас при возникновении спорных ситуаций с клиентом.

{% endnote %}

## Период проката

Указание сроков аренды — обязательный этап оформления, от которого зависит расчёт стоимости.

- **Дата и время выдачи** — фиксируется момент передачи инструмента клиенту.
- **Плановая дата возврата** — ожидаемый срок возврата оборудования.
- **Тип периода** — почасовой, посуточный, понедельный или произвольный.
- **Продление** — возможность продлить срок проката из карточки действующего заказа.

## Залог

Залоговый платёж является гарантией сохранности оборудования и фиксируется при оформлении заказа.

- **Сумма залога** — рассчитывается автоматически на основе залоговой стоимости выбранных позиций.
- **Способ оплаты** — наличные, карта, безналичный расчёт.
- **Квитанция о залоге** — возможность распечатать подтверждение приёма залога.
- **Частичный залог** — при необходимости сумму залога можно скорректировать вручную.

{% note info %}

Сумма залога по умолчанию определяется настройками каждой позиции в каталоге. Администратор может задать политику залоговых платежей в разделе [Настройки](../settings/).

{% endnote %}

## Расчёт стоимости

Стоимость проката рассчитывается автоматически на основе выбранных позиций, тарифов и срока аренды.

- **Автоматический расчёт** — система подбирает оптимальный тариф в зависимости от длительности проката.
- **Ручная корректировка** — оператор может изменить рассчитанную стоимость при необходимости.
- **Скидки** — применение фиксированных или процентных скидок к итоговой сумме.
- **Итоговая стоимость** — отображается общая сумма к оплате с разбивкой по позициям.

<!-- screenshot: блок расчёта стоимости с разбивкой по позициям -->

## Оплата

Фиксация оплаты за прокат может быть выполнена при оформлении или при возврате инструмента.

- **Предоплата** — полная или частичная оплата при оформлении заказа.
- **Оплата при возврате** — расчёт по фактическому сроку использования.
- **Способы оплаты** — наличные, банковская карта, безналичный расчёт.
- **Чек или квитанция** — формирование документа об оплате.

## Сохранение и выдача

После заполнения формы заказ сохраняется и инструмент переводится в статус «В прокате».

- **Сохранение заказа** — заказ создаётся со статусом «Оформлен».
- **Печать договора** — автоматическое предложение напечатать договор проката.
- **Обновление статусов** — выбранные позиции автоматически переводятся в статус «В прокате».
- **Переход к карточке** — после сохранения открывается полная карточка заказа.
