# Оформление проката

Оформление проката — основная операция модуля, в ходе которой фиксируется выдача инструмента клиенту. Форма заказа собирает информацию о клиенте, инструменте, сроках и условиях аренды.

<!-- screenshot: кнопка создания нового проката на панели инструментов -->

## Открытие формы оформления

Новый заказ на прокат можно создать из раздела проката, нажав кнопку «Новый прокат».

<!-- screenshot: форма оформления нового проката -->

## Выбор клиента

При оформлении проката необходимо указать клиента, которому выдаётся оборудование. Система поддерживает поиск по существующей базе и создание нового клиента.

- **Поиск существующего клиента** — по имени или телефону.
- **Создание нового клиента** — если клиент ещё не зарегистрирован в системе.
- **Паспортные данные** — при оформлении проката указываются паспортные данные клиента (дата рождения, серия и номер паспорта, кем и когда выдан, адрес регистрации и фактический адрес). Эти данные сохраняются в заказе и подставляются в договор.

{% note warning %}

Оформление проката невозможно для клиента, находящегося в чёрном списке. Система заблокирует создание заказа.

{% endnote %}

## Выбор инструмента

Каждый заказ оформляется на одну единицу инструмента.

- **Поиск по каталогу** — поиск по модели, категории или инвентарному номеру.
- **Фильтрация по доступности** — отображаются только инструменты со статусом «Рабочий».
- **Комплектация** — описание комплектации фиксируется в поле «Оборудование» заказа.

<!-- screenshot: выбор инструмента из каталога -->

{% note info %}

Один заказ на прокат — один инструмент. Для выдачи нескольких единиц создайте отдельный заказ на каждую.

{% endnote %}

## Период проката

Указание сроков аренды — обязательный этап оформления.

- **Дата выдачи** — фиксируется момент передачи инструмента клиенту.
- **Плановая дата возврата** — ожидаемый срок возврата оборудования.
- **Количество дней** — рассчитывается автоматически на основе дат.

## Расчёт стоимости

Стоимость проката рассчитывается на основе суточного тарифа инструмента и количества дней.

- **Суточный тариф** — подставляется из карточки инструмента, может быть скорректирован.
- **Итоговая стоимость** — суточный тариф × количество дней.
- **Предоплата** — фиксация внесённой предоплаты.

<!-- screenshot: блок расчёта стоимости -->

## Сохранение и выдача

После заполнения формы заказ сохраняется и инструмент переводится в статус «В прокате».

- **Сохранение заказа** — заказ создаётся со статусом «В прокате».
- **Печать договора** — из карточки заказа можно сформировать и распечатать договор проката.
- **Обновление статуса инструмента** — выбранный инструмент автоматически переводится в статус «В прокате».
