# Договор проката

Договор проката — документ, регулирующий отношения между сервисным центром и клиентом при выдаче оборудования во временное пользование. Система формирует договор в формате PDF из загруженного DOCX-шаблона.

<!-- screenshot: пример сформированного договора проката -->

## Автоматическое формирование

Договор проката генерируется на основе данных заказа. Все поля заполняются из карточки проката автоматически.

- **Данные клиента** — ФИО, паспортные данные, контактная информация подставляются из заказа.
- **Инструмент** — модель, серийный и инвентарный номер, комплектация.
- **Условия проката** — сроки, суточный тариф, итоговая стоимость.
- **Данные организации** — название, реквизиты и контакты из настроек.

{% note info %}

Договор формируется из карточки заказа на прокат. При изменении данных заказа можно сформировать обновлённый договор.

{% endnote %}

## Шаблон договора

Содержание и оформление договора определяются DOCX-шаблоном, который загружается администратором.

- **Загрузка шаблона** — администратор загружает DOCX-файл с переменными-плейсхолдерами.
- **Переменные** — в шаблоне используются переменные вида `${field}`, которые заменяются данными заказа.
- **Один шаблон** — для договора проката загружается один шаблон на организацию.

### Доступные переменные

| Переменная | Описание |
|---|---|
| `${id}` | Номер заказа |
| `${started_at}` | Дата начала проката |
| `${expected_end_at}` | Плановая дата возврата |
| `${price_per_day}` | Суточный тариф |
| `${price_per_day_words}` | Суточный тариф прописью |
| `${equipment}` | Комплектация |
| `${customer.*}` | Данные клиента |
| `${tool.*}` | Данные инструмента |
| `${branch.*}` | Данные филиала |
| `${passport.*}` | Паспортные данные |

{% note info %}

Настройка шаблона договора доступна в разделе [Шаблоны документов](../settings/templates.md). Рекомендуется согласовать текст договора с юристом перед началом использования.

{% endnote %}

## Печать договора

Печать доступна из карточки заказа на прокат.

- **Генерация PDF** — система заполняет DOCX-шаблон данными заказа и формирует PDF-файл.
- **Предварительный просмотр** — просмотр сформированного документа.
- **Скачивание** — документ можно скачать в формате PDF.

## Процесс возврата

Возврат инструмента — завершающий этап проката.

- **Фиксация фактической даты возврата** — система рассчитывает итоговую стоимость по фактическому сроку использования.
- **Закрытие заказа** — заказ переводится в статус «Принят», инструмент — в статус «Рабочий».
