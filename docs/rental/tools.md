# Управление инструментом

Каталог инструмента — основа модуля проката. Здесь ведётся учёт всего оборудования, доступного для сдачи в аренду.

<!-- screenshot: каталог инструмента с фильтрами и списком позиций -->

## Каталог инструмента

Каталог содержит полный перечень оборудования, которое может быть выдано в прокат. Каждая позиция описывается набором характеристик.

- **Модель** — наименование и модель инструмента.
- **Категория** — группировка по типу оборудования.
- **Бренд** — производитель инструмента.
- **Серийный номер** — уникальный серийный номер единицы.
- **Инвентарный номер** — внутренний идентификатор единицы оборудования.
- **Фотографии** — изображения инструмента для идентификации.

<!-- screenshot: карточка инструмента с заполненными полями -->

## Добавление новых позиций

Для пополнения каталога предусмотрена форма добавления.

- **Основные параметры** — модель, категория, бренд, серийный и инвентарный номер.
- **Суточный тариф** — стоимость проката за сутки.
- **Рыночная стоимость** — стоимость инструмента для учёта.
- **Филиал** — привязка инструмента к конкретному филиалу.
- **Заметки** — дополнительная информация.

{% note info %}

Категории и бренды инструмента настраиваются администратором в разделе [Настройки](../settings/).

{% endnote %}

## Статусы инструмента

Каждая единица оборудования имеет текущий статус, отражающий её доступность. Статус обновляется автоматически при операциях проката.

- **Рабочий** — инструмент находится на складе и готов к выдаче.
- **В прокате** — инструмент выдан клиенту.
- **На обслуживании** — инструмент отправлен на техническое обслуживание или ремонт.
- **Сломан** — инструмент неисправен.
- **Украден** — инструмент отмечен как утраченный.

<!-- screenshot: список инструмента с цветовой индикацией статусов -->

{% note warning %}

Инструмент, недоступный для проката (сломан, на обслуживании, украден), не отображается в списке при оформлении нового проката.

{% endnote %}

## Отслеживание местоположения

Система позволяет отслеживать, у какого клиента находится каждая единица инструмента и когда она должна быть возвращена.

- **Текущий клиент** — информация о клиенте, у которого находится инструмент.
- **Дата выдачи и плановый возврат** — сроки текущей аренды.

## Тариф

Стоимость проката задаётся как суточный тариф для каждой единицы инструмента.

- **Суточный тариф** — фиксированная стоимость за сутки проката.
- Итоговая стоимость рассчитывается как суточный тариф, умноженный на количество дней аренды.

{% note info %}

Один заказ на прокат оформляется на одну единицу инструмента. Если клиенту нужно несколько инструментов, создаётся отдельный заказ на каждый.

{% endnote %}
