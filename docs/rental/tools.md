# Управление инструментом

Каталог инструмента — основа модуля проката. Здесь ведётся учёт всего оборудования, доступного для сдачи в аренду: от ручного инструмента до крупной строительной техники.

<!-- screenshot: каталог инструмента с фильтрами и списком позиций -->

## Каталог инструмента

Каталог содержит полный перечень оборудования, которое может быть выдано в прокат. Каждая позиция описывается набором характеристик.

- **Название и описание** — наименование инструмента и его краткое описание.
- **Категория** — группировка по типу оборудования (электроинструмент, ручной инструмент, строительная техника и т.д.).
- **Инвентарный номер** — уникальный идентификатор единицы оборудования.
- **Фотографии** — визуальное представление инструмента для удобства идентификации.

<!-- screenshot: карточка инструмента с заполненными полями -->

## Добавление новых позиций

Для пополнения каталога предусмотрена форма добавления с обязательными и дополнительными полями.

- **Основные параметры** — название, категория, инвентарный номер и описание.
- **Тарификация** — стоимость проката за час, сутки, неделю или другой период.
- **Залоговая стоимость** — сумма залога, которую необходимо внести при получении инструмента.
- **Балансовая стоимость** — стоимость инструмента для учёта износа и компенсации при повреждении.
- **Комплектация** — перечень элементов, входящих в комплект при выдаче.

{% note info %}

Категории инструмента настраиваются администратором в разделе [Настройки](../settings/). Вы можете создать любую структуру категорий, подходящую для вашего бизнеса.

{% endnote %}

## Статусы инструмента

Каждая единица оборудования имеет текущий статус, отражающий её доступность. Статус обновляется автоматически при операциях проката.

- **Доступен** — инструмент находится на складе и готов к выдаче.
- **В прокате** — инструмент выдан клиенту, указана плановая дата возврата.
- **На обслуживании** — инструмент отправлен на техническое обслуживание или ремонт.
- **Списан** — инструмент выведен из оборота и не доступен для проката.

<!-- screenshot: список инструмента с цветовой индикацией статусов -->

{% note warning %}

Инструмент в статусе «На обслуживании» не отображается в списке доступных при оформлении нового проката. Не забывайте переводить инструмент обратно в статус «Доступен» после завершения обслуживания.

{% endnote %}

## Отслеживание местоположения

Система позволяет отслеживать, у какого клиента находится каждая единица инструмента и когда она должна быть возвращена.

- **Текущий клиент** — информация о клиенте, у которого находится инструмент.
- **Дата выдачи и плановый возврат** — сроки текущей аренды.
- **Просроченные возвраты** — автоматическая подсветка позиций, не возвращённых в срок.

## История использования

По каждой единице инструмента ведётся полная история операций.

- **Журнал выдач** — кому, когда и на какой срок выдавался инструмент.
- **Фиксация повреждений** — записи о выявленных повреждениях при возврате.
- **История обслуживания** — даты и описание проведённых технических работ.

<!-- screenshot: история использования конкретного инструмента -->

## Инвентаризация

Периодическая сверка фактического наличия инструмента с данными в системе.

- **Создание инвентаризации** — запуск процедуры сверки по всему каталогу или отдельной категории.
- **Фиксация фактического наличия** — отметка каждой единицы как «обнаружена» или «отсутствует».
- **Акт расхождений** — автоматическое формирование документа с выявленными расхождениями.

{% note info %}

Рекомендуется проводить инвентаризацию не реже одного раза в месяц. Это позволяет своевременно выявлять потери и контролировать состояние оборудования.

{% endnote %}

## Тарифы и ценообразование

Гибкая система тарификации позволяет задать стоимость проката для различных периодов.

- **Почасовой тариф** — стоимость за каждый час использования.
- **Суточный тариф** — фиксированная стоимость за сутки проката.
- **Недельный и месячный тарифы** — скидочные тарифы при длительной аренде.
- **Индивидуальные условия** — возможность задать особые условия для конкретного клиента или заказа.
