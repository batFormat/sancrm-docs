# Канбан-доска

Канбан-доска — визуальный инструмент для управления ремонтами. Каждый столбец представляет статус, а карточки — отдельные заказы. Перетаскивание карточки между столбцами меняет статус ремонта.

<!-- screenshot: канбан-доска с ремонтами, распределёнными по столбцам-статусам -->

## Общий вид доски

Доска отображает ремонты в виде карточек, сгруппированных по столбцам-статусам. Это даёт мгновенное представление о загрузке сервисного центра и распределении заказов по этапам.

- Каждый столбец соответствует группе статусов ремонта (Новые, Диагностика, К выполнению, В работе, На удержании, Готовые, Отказ, Закрыт).
- На карточке отображается ключевая информация: номер заказа, клиент, устройство, исполнитель.

## Drag-and-drop

Перетаскивание карточки между столбцами — основной способ смены статуса ремонта на канбан-доске.

- Перетащите карточку в нужный столбец для смены статуса.
- При перетаскивании подсвечиваются только допустимые столбцы (согласно правилам переходов).
- Если для перехода требуется заполнение обязательных полей, откроется диалоговое окно.

{% note info %}

Drag-and-drop соблюдает все настроенные правила переходов и ограничения по ролям. Нельзя перетащить карточку в статус, переход в который запрещён для вашей роли.

{% endnote %}

## Столбцы доски

Столбцы на канбан-доске соответствуют группам статусов (RepairGroup). Все статусы одной группы отображаются в одном столбце. Порядок столбцов фиксирован и отражает логику рабочего процесса.

## Фильтрация на доске

На канбан-доске доступны фильтры для отображения только нужных ремонтов.

- **По исполнителю** — показать ремонты конкретного мастера.
- **По филиалу** — при работе с несколькими филиалами.

## Карточка ремонта на доске

Каждая карточка содержит компактную сводку по ремонту. Состав отображаемой информации можно настроить.

<!-- screenshot: карточка ремонта на канбан-доске крупным планом -->

- **Номер заказа** — уникальный идентификатор ремонта.
- **Клиент** — имя и телефон клиента.
- **Устройство** — тип и модель устройства.
- **Исполнитель** — назначенный мастер.
- **Статус** — текущий статус ремонта внутри группы.

## Быстрые действия

С карточки на доске можно выполнить основные действия, не открывая полную карточку ремонта.

- Смена статуса через контекстное меню.
- Назначение или смена исполнителя.
- Добавление комментария.
- Переход к полной карточке ремонта.

{% note warning %}

На мобильных устройствах drag-and-drop может быть недоступен. Используйте контекстное меню карточки для смены статуса.

{% endnote %}

## Автообновление

Доска обновляется в реальном времени — изменения, внесённые другими пользователями, отображаются без перезагрузки страницы.
