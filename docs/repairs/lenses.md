# Фильтры и представления

Фильтры и предустановленные представления (линзы) позволяют быстро находить нужные ремонты в общем списке. Линзы — это готовые наборы фильтров для типовых сценариев работы.

<!-- screenshot: список ремонтов с панелью фильтров -->

## Панель фильтров

Панель фильтров расположена над списком ремонтов и позволяет отобрать заказы по различным критериям.

- **Статус** — отображение ремонтов в одном или нескольких статусах.
- **Исполнитель** — ремонты конкретного мастера.
- **Филиал** — при наличии нескольких филиалов.

<!-- screenshot: развёрнутая панель фильтров с заполненными полями -->

## Предустановленные линзы

Система содержит набор предустановленных линз — готовых фильтров для типовых задач. Линзы доступны в меню над списком ремонтов.

- **Закрытые ремонты** — ремонты в финальных статусах.
- **Повторное уведомление** — ремонты, по которым необходимо повторно уведомить клиента.
- **Без модели** — ремонты, в которых не указана модель устройства.
- **Ожидают запчасти** — ремонты в статусах, связанных с заказом запчастей.
- **Согласованные** — ремонты, прошедшие согласование стоимости.
- **Без мастера** — ремонты, которым не назначен исполнитель.
- **На согласовании** — ремонты в статусе «Согласование».

{% note info %}

Линзы являются предустановленными представлениями и не могут быть созданы или изменены пользователями. Для фильтрации по произвольным критериям используйте панель фильтров.

{% endnote %}

## Поиск по списку

В списке ремонтов доступен поиск по номеру заказа и другим ключевым полям.

- Результаты обновляются по мере ввода.
- Поиск работает совместно с активными фильтрами и линзами.

## Сортировка

Список ремонтов можно отсортировать по видимым полям таблицы.

- Сортировка по возрастанию или убыванию.
- По умолчанию ремонты сортируются по дате создания.
