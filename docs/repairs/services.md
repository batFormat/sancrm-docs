# Услуги

Услуги — это работы и операции, выполняемые мастером в рамках ремонта. Добавление услуг формирует стоимость работ и обеспечивает прозрачность расчётов для клиента.

<!-- screenshot: вкладка «Услуги» в карточке ремонта -->

## Добавление услуг к ремонту

Услуги добавляются из прайс-листа или создаются вручную непосредственно в карточке ремонта.

- **Выбор из прайс-листа** — поиск по названию или категории услуги.
- **Ручной ввод** — создание произвольной позиции с указанием названия и цены.
- **Количество** — указание количества, если услуга применяется многократно.

<!-- screenshot: диалог добавления услуги из прайс-листа -->

## Прайс-лист услуг

Прайс-лист содержит каталог стандартных услуг с фиксированными ценами. Это ускоряет оформление и обеспечивает единообразие расценок.

- **Базовая цена** — цена по умолчанию, которая подставляется при добавлении услуги.
- **Активность** — услугу можно деактивировать, чтобы она не отображалась при добавлении.
- **Порядок** — услуги отображаются в настраиваемом порядке.

{% note info %}

Управление прайс-листом описано в разделе [Настройки](../settings/). Администратор может создавать, редактировать и деактивировать услуги.

{% endnote %}

## Формирование стоимости

Стоимость работ по ремонту складывается из всех добавленных услуг. Система автоматически рассчитывает итоговую сумму.

- **Автоматический расчёт** — итоговая стоимость услуг пересчитывается при каждом изменении.
- **Ручная корректировка цены** — для конкретного ремонта цена услуги может быть изменена.
- **Скидки** — скидка на отдельную услугу или на все услуги в ремонте.
- **Итог по ремонту** — общая стоимость = стоимость услуг + стоимость запчастей.

## Исполнитель услуги

Каждой услуге в ремонте можно назначить конкретного исполнителя.

- **Назначение мастера** — выбор исполнителя для каждой услуги отдельно.
- **Распределение работ** — разные услуги в одном ремонте могут быть выполнены разными мастерами.

<!-- screenshot: назначение исполнителя для услуги в карточке ремонта -->

{% note warning %}

При изменении цены услуги в прайс-листе ранее оформленные ремонты сохраняют прежнюю стоимость. Изменение цены распространяется только на новые заказы.

{% endnote %}
