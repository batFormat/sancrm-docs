# Запчасти и товары

Добавление запчастей к ремонту позволяет вести точный учёт использованных материалов, рассчитывать себестоимость и автоматически списывать позиции со склада.

<!-- screenshot: вкладка «Запчасти» в карточке ремонта -->

## Добавление запчастей к ремонту

Запчасти добавляются к ремонту из складского каталога. Система обеспечивает поиск, проверку наличия и автоматический расчёт стоимости.

- **Поиск по каталогу** — поиск по названию, артикулу или штрихкоду.
- **Выбор склада** — если складов несколько, выбирается конкретный склад списания.
- **Указание количества** — количество единиц запчасти для данного ремонта.

<!-- screenshot: диалог добавления запчасти с поиском по каталогу -->

## Цены и наценки

Система поддерживает гибкую модель ценообразования для запчастей, добавляемых к ремонту.

- **Закупочная цена** — себестоимость запчасти, подставляется автоматически из складского учёта.
- **Цена продажи** — цена для клиента, рассчитывается автоматически по заданной наценке.
- **Ручная корректировка** — возможность изменить цену вручную для конкретного ремонта.
- **Скидка** — применение скидки на отдельные позиции или на все запчасти.

{% note info %}

Правила наценок и скидок настраиваются в разделе [Настройки](../settings/). Можно задать разные наценки для разных категорий товаров.

{% endnote %}

## Резервирование и списание

При добавлении запчасти к ремонту система может зарезервировать её на складе, а при завершении ремонта — списать.

- **Резервирование** — запчасть остаётся на складе, но учитывается как зарезервированная.
- **Списание** — фактическое уменьшение остатка на складе, обычно при смене статуса на «Готов».
- **Возврат** — если запчасть не потребовалась, резерв снимается.

{% note warning %}

Если на момент списания запчасти нет в наличии, система предупредит об этом. Проверяйте остатки перед завершением ремонта.

{% endnote %}

## Связь со складом

Модуль запчастей в ремонтах тесно интегрирован со складским учётом.

- Все операции с запчастями отражаются в складских движениях.
- Из карточки ремонта можно перейти к складской карточке товара.
- При недостатке запчасти можно сформировать заявку на закупку.

<!-- screenshot: связь позиции в ремонте со складской карточкой товара -->

## Итоговая стоимость запчастей

Система автоматически рассчитывает общую стоимость всех запчастей в ремонте.

- Сумма отображается в карточке ремонта и учитывается в итоговой стоимости заказа.
- Отдельно показываются себестоимость и прибыль по запчастям.
- При изменении состава или цен итог пересчитывается автоматически.

## Часто используемые запчасти

Для ускорения работы можно настроить список часто используемых запчастей, доступный при добавлении.

- Быстрый доступ к популярным позициям без поиска по каталогу.
- Списки могут формироваться автоматически на основе статистики использования.
