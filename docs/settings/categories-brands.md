# Категории и бренды

Справочники категорий и брендов устройств используются при создании ремонтов для классификации принимаемой техники. Правильно настроенные справочники ускоряют оформление заказов и улучшают аналитику.

<!-- screenshot: страница настроек категорий и брендов -->

## Категории устройств

Категории позволяют классифицировать устройства по типу: смартфоны, ноутбуки, планшеты, бытовая техника и другие. Категории используются для фильтрации ремонтов и построения отчётов.

### Создание категории

Для добавления новой категории укажите её название и, при необходимости, родительскую категорию. Система поддерживает вложенную иерархию произвольной глубины.

- **Название** — отображаемое имя категории (например, «Смартфоны», «Ноутбуки»).
- **Родительская категория** — категория верхнего уровня для группировки (например, «Мобильные устройства»).

<!-- screenshot: форма создания категории с выбором родительской категории -->

### Иерархия категорий

Категории могут быть организованы в древовидную структуру. Это позволяет создать детализированную классификацию — например, «Техника > Мобильные устройства > Смартфоны > Android-смартфоны».

<!-- screenshot: древовидная структура категорий -->

{% note info %}

При выборе категории в карточке ремонта отображается полный путь в иерархии, что помогает точно классифицировать устройство.

{% endnote %}

### Редактирование и удаление

Категорию можно переименовать или переместить в другую ветку иерархии. Удаление категории возможно только в том случае, если к ней не привязаны ремонты.

## Бренды

Справочник брендов содержит список производителей устройств: Apple, Samsung, Xiaomi и другие. Бренд выбирается при оформлении ремонта вместе с моделью устройства.

### Добавление бренда

Для добавления нового бренда достаточно указать его название. Дополнительно можно загрузить логотип бренда для визуального отображения в интерфейсе.

<!-- screenshot: форма добавления бренда -->

### Привязка к категориям

Бренды можно связать с определёнными категориями устройств. При выборе категории в карточке ремонта список брендов будет автоматически отфильтрован — отобразятся только релевантные производители.

- Например, бренд «Apple» можно привязать к категориям «Смартфоны», «Планшеты» и «Ноутбуки».
- Если привязка не настроена, бренд доступен для выбора во всех категориях.

<!-- screenshot: привязка бренда к категориям через чекбоксы -->

{% note warning %}

Удаление бренда, привязанного к существующим ремонтам, невозможно. Если бренд больше не актуален, скройте его — он перестанет отображаться в выпадающих списках, но сохранится в истории.

{% endnote %}

## Модели устройств

Для часто ремонтируемых устройств можно вести справочник моделей. Модель привязывается к бренду и категории, что позволяет быстро заполнять карточку ремонта.
