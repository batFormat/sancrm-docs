# Категории и бренды

Справочники категорий и брендов используются при создании ремонтов и управлении каталогом инструмента для проката. Правильно настроенные справочники ускоряют оформление заказов.

<!-- screenshot: страница настроек категорий и брендов -->

## Категории

Категории позволяют классифицировать устройства и инструменты. Они используются при создании ремонтов, в каталоге инструмента для проката и для фильтрации.

### Создание категории

Для добавления новой категории укажите её название и, при необходимости, описание.

- **Название** — отображаемое имя категории (например, «Электроинструмент», «Бензоинструмент»).
- **Описание** — дополнительное описание категории.
- **Класс инструмента** — для проката: электроинструмент или бензоинструмент.
- **Порядок сортировки** — позиция в списке.

{% note info %}

Категории имеют плоскую структуру без вложенности. Каждая категория принадлежит вашей организации и не видна другим аккаунтам.

{% endnote %}

## Бренды

Справочник брендов содержит список производителей. Бренд выбирается при создании инструмента для проката и при оформлении ремонта.

### Добавление бренда

Для добавления нового бренда укажите его название и, при необходимости, описание.

- **Название** — имя бренда (например, «Bosch», «Makita», «Stihl»).
- **Описание** — дополнительное описание.
- **Порядок сортировки** — позиция в списке.

<!-- screenshot: форма добавления бренда -->

{% note info %}

Бренды и категории — независимые справочники. Привязка бренда к конкретным категориям не поддерживается — все бренды доступны при выборе во всех категориях.

{% endnote %}
