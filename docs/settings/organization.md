# Настройки организации

Раздел настроек организации позволяет задать основные сведения о вашей компании, которые используются в документах, уведомлениях и интерфейсе системы.

<!-- screenshot: страница настроек организации — общий вид -->

## Название и описание компании

Укажите название вашей организации, которое будет отображаться в шапке системы и в печатных документах. Дополнительно можно добавить краткое описание деятельности.

<!-- screenshot: поля ввода названия и описания компании -->

## Логотип

Загрузите логотип компании, который будет использоваться в квитанциях, гарантийных талонах и других печатных формах. Рекомендуемый формат — PNG с прозрачным фоном.

{% note info %}

Оптимальный размер логотипа — не менее 200x200 пикселей. Слишком маленькие изображения могут выглядеть размыто при печати.

{% endnote %}

<!-- screenshot: область загрузки логотипа с предпросмотром -->

## Юридические данные

Заполните юридическую информацию об организации: ИНН, ОГРН, юридический адрес и банковские реквизиты. Эти данные подставляются в шаблоны документов через переменные.

- **ИНН** — идентификационный номер налогоплательщика.
- **ОГРН** — основной государственный регистрационный номер.
- **Юридический адрес** — адрес регистрации организации.
- **Банковские реквизиты** — расчётный счёт, БИК и наименование банка.

<!-- screenshot: форма юридических данных с заполненными полями -->

## Контактные данные

Укажите основные контактные данные организации: телефон, email и адрес веб-сайта. Контакты могут использоваться в печатных документах и уведомлениях клиентам.

<!-- screenshot: блок контактных данных -->

## Часовой пояс

Выберите часовой пояс, в котором работает ваша организация. Часовой пояс влияет на отображение дат и времени во всей системе, а также на расчёт сроков ремонта.

{% note warning %}

Изменение часового пояса затрагивает отображение времени для всех пользователей организации. Рекомендуется установить корректный часовой пояс при первоначальной настройке и не менять его без необходимости.

{% endnote %}

<!-- screenshot: выпадающий список выбора часового пояса -->

## Валюта

Укажите основную валюту, в которой ведётся расчёт стоимости ремонтов, товаров и услуг. Символ валюты отображается в карточках ремонтов, складских документах и отчётах.
