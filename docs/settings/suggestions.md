# Справочники подсказок

Справочники подсказок содержат предзаполненные варианты для полей карточки ремонта. Подсказки ускоряют оформление заказов и стандартизируют заполнение данных.

<!-- screenshot: страница управления справочниками подсказок -->

## Типы справочников

Система содержит четыре типа справочников подсказок.

### Проблемы

Справочник описаний неисправностей, которые предлагаются при заполнении поля «Заявленная неисправность» в карточке ремонта.

- Примеры: «Не включается», «Не заряжается», «Не заводится», «Вибрация при работе».
- Подсказки поддерживают иерархию — можно создать дочерние записи для детализации.

### Причины

Справочник причин неисправности, используемый при фиксации результата диагностики.

- Примеры: «Износ подшипников», «Обрыв обмотки», «Засорение карбюратора».

### Услуги

Справочник стандартных услуг с описаниями для подсказок при добавлении работ к ремонту.

### Оборудование

Справочник типов оборудования для быстрого заполнения информации об устройстве.

## Классификация по типу инструмента

Подсказки можно привязать к классу инструмента (электроинструмент или бензоинструмент). При заполнении карточки ремонта система будет предлагать только релевантные подсказки в зависимости от типа ремонтируемого устройства.

{% note info %}

Справочники подсказок поддерживают иерархию — можно создавать дочерние записи через поле «Родительский элемент». Это позволяет группировать подсказки по темам.

{% endnote %}

## Добавление записей

Для добавления новой подсказки укажите текст и, при необходимости, привяжите к классу инструмента и родительскому элементу.
