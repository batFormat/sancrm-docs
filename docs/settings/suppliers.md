# Поставщики

Справочник поставщиков используется для учёта контрагентов, у которых вы закупаете запчасти и товары. Привязка поставщика к документам поступления позволяет отслеживать историю закупок.

<!-- screenshot: список поставщиков в настройках -->

## Добавление поставщика

Для регистрации нового поставщика заполните карточку с основной информацией.

- **Название** — наименование компании-поставщика или ФИО частного лица.
- **Телефон** — контактный номер.
- **Email** — адрес электронной почты.
- **Адрес** — физический адрес поставщика.
- **Комментарий** — произвольные заметки (условия доставки, скидки, особенности работы).

<!-- screenshot: форма создания нового поставщика -->

## Привязка к поступлениям

При создании документа поступления товаров выберите поставщика из справочника. Это позволяет отслеживать, от какого поставщика получен каждый товар.

{% note info %}

Поставщик указывается при создании документа прихода. В карточке поставщика можно просмотреть связанные документы.

{% endnote %}

## Редактирование и удаление

Информацию о поставщике можно изменить в любой момент. Обновлённые данные отобразятся в карточке, но не затронут ранее созданные документы поступления.

{% note warning %}

Удаление поставщика невозможно, если к нему привязаны документы поступления. Если поставщик больше не актуален, добавьте соответствующую пометку в комментарий.

{% endnote %}
