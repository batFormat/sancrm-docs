# Шаблоны документов

Шаблоны документов определяют содержание и оформление печатных форм — квитанций приёма и договоров проката. Шаблоны загружаются в формате DOCX с переменными-плейсхолдерами.

<!-- screenshot: список шаблонов документов -->

## Типы шаблонов

Система поддерживает три типа документов.

- **Квитанция приёма** — документ, подтверждающий приём устройства в ремонт. Полная версия с подробной информацией.
- **Квитанция краткая** — компактная версия приёмной квитанции.
- **Договор аренды** — договор проката инструмента.

Для каждого типа загружается один шаблон.

<!-- screenshot: карточки типов шаблонов -->

## Загрузка шаблона

Шаблон — это DOCX-файл, содержащий текст документа с переменными. Для загрузки или замены шаблона:

- Подготовьте DOCX-файл с нужным форматированием и переменными.
- Загрузите файл через интерфейс настроек.
- При генерации документа система заполнит переменные данными из заказа и сформирует PDF.

{% note info %}

Визуальный редактор шаблонов отсутствует. Подготовьте DOCX-файл в любом текстовом редакторе (Microsoft Word, LibreOffice Writer) и загрузите его в систему.

{% endnote %}

## Переменные

Для подстановки динамических данных в шаблон используются переменные вида `${field}`. При генерации документа переменные автоматически заменяются реальными значениями.

### Переменные для квитанции ремонта

Переменные заполняются из карточки ремонта, данных клиента, устройства и организации.

### Переменные для договора аренды

| Переменная | Описание |
|---|---|
| `${id}` | Номер заказа |
| `${started_at}` | Дата начала проката |
| `${expected_end_at}` | Плановая дата возврата |
| `${price_per_day}` | Суточный тариф |
| `${price_per_day_words}` | Суточный тариф прописью |
| `${equipment}` | Комплектация |

## Генерация и печать

При нажатии кнопки печати в карточке заказа система заполняет DOCX-шаблон данными и формирует PDF-файл. Документ открывается для предпросмотра и печати.

{% note warning %}

После изменения шаблона все новые документы будут формироваться по обновлённой версии. Ранее сгенерированные PDF-файлы не изменяются.

{% endnote %}
