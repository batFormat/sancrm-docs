# Склады

Настройка складов определяет, где хранятся запчасти и товары вашего сервисного центра. Остатки отслеживаются раздельно по каждому складу.

<!-- screenshot: список складов в настройках -->

## Создание склада

Для добавления нового склада укажите его название. Название склада используется в складских документах, при списании запчастей на ремонт и в отчётах.

- **Название** — понятное имя склада (например, «Основной склад», «Склад запчастей»).
- **Адрес** — физический адрес склада (при необходимости).

<!-- screenshot: форма создания нового склада -->

## Склад по умолчанию

Один из складов можно назначить складом по умолчанию. Он будет автоматически подставляться при создании приходных документов и при списании запчастей на ремонт.

{% note info %}

При необходимости сотрудник может вручную выбрать другой склад в момент операции.

{% endnote %}

## Доступ к складам

Сотрудники получают доступ к складам через настройку привязки. Администратор назначает каждому пользователю набор доступных складов.

- Один сотрудник может иметь доступ к нескольким складам.
- Привязка сотрудников к складам выполняется через настройки пользователей.

## Перемещение между складами

Если у вас несколько складов, доступна функция перемещения товаров между ними. Перемещение фиксируется документом и отражается в остатках обоих складов.

{% note warning %}

Удаление склада невозможно, если на нём числятся остатки или к нему привязаны складские документы. Сначала переместите все товары и завершите открытые документы.

{% endnote %}
