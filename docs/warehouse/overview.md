# Обзор складского модуля

Складской модуль SanCRM обеспечивает полный цикл учёта товаров — от поступления на склад до продажи или списания. Раздел описывает основные возможности, принципы работы и структуру модуля.

## Назначение модуля

Модуль «Склад» предназначен для ведения складского учёта в сервисных центрах и ремонтных мастерских. Он позволяет контролировать остатки, фиксировать все движения товаров и формировать необходимые документы.

<!-- screenshot: главная страница раздела «Склад» -->

## Мультисклад

Система поддерживает работу с несколькими складами одновременно. Каждый склад имеет собственные остатки, и вы можете отслеживать наличие товаров по каждому складу отдельно.

- Создание и настройка складов.
- Назначение склада по умолчанию для филиала.
- Просмотр остатков в разрезе складов.

{% note info %}

Количество складов не ограничено. Вы можете создать отдельные склады для запчастей, готовой продукции, витрины и т. д.

{% endnote %}

## Учёт остатков

Остатки товаров обновляются автоматически при проведении складских документов. Вы всегда видите актуальное количество по каждой позиции.

- Текущие остатки по складу.
- История движения товара.
- Контроль минимальных остатков и оповещение «Мало на складе».

<!-- screenshot: таблица остатков с фильтрацией по складу -->

## Складские операции

Все операции с товарами оформляются через складские документы. Каждый документ фиксирует тип операции, дату, ответственного сотрудника и список позиций.

- **Приход** — поступление товаров от поставщика.
- **Списание** — удаление товаров с остатков.
- **Перемещение** — передача товаров между складами.
- **Инвентаризация** — сверка фактических и учётных остатков.
- **Возврат поставщику** — оформление возврата ранее принятых товаров.

## Интеграция с другими модулями

Складской модуль тесно связан с остальными разделами системы. Товары со склада используются при оформлении заказов, продажах и ремонтах.

- Автоматическое списание запчастей при добавлении в ремонт.
- Подбор запчастей из остатков при оформлении ремонта.
- Очередь «Под заказ» — товары, ожидающие поступления на склад.
- Учёт себестоимости в отчётах.
