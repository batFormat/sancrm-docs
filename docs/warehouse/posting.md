# Приход

Документ прихода фиксирует поступление товаров на склад от поставщика. При проведении прихода остатки на выбранном складе увеличиваются, а закупочные цены обновляются.

## Когда используется

Приходный документ создаётся при получении товаров от поставщика — будь то плановая поставка или разовая закупка. Документ позволяет зафиксировать, что именно поступило, в каком количестве и по какой цене.

## Создание документа прихода

Для создания прихода перейдите в раздел складских документов и выберите тип «Приход». Заполните основные поля и добавьте товарные позиции.

<!-- screenshot: форма создания документа прихода -->

### Основные поля

- **Склад** — на какой склад поступают товары.
- **Поставщик** — контрагент, от которого получены товары (необязательно).
- **Комментарий** — дополнительная информация о поставке.

{% note info %}

Если поставщик ещё не добавлен в систему, его можно создать прямо из формы прихода, не покидая страницу.

{% endnote %}

## Выбор поставщика

Поставщик выбирается из справочника контрагентов. Вы можете найти поставщика по названию, ИНН или телефону. Привязка к поставщику позволяет отслеживать историю поставок.

- Поиск по справочнику контрагентов.
- Быстрое создание нового поставщика.
- История поставок от выбранного поставщика.

## Добавление товарных позиций

В документ прихода добавляются товары с указанием количества и закупочной цены.

- Поиск товара по наименованию, артикулу или штрихкоду.
- Указание количества и закупочной цены.
- Создание нового товара прямо из документа прихода.

<!-- screenshot: таблица позиций в документе прихода -->

{% note warning %}

Закупочная цена, указанная в приходе, обновляет себестоимость товара. Убедитесь, что цена указана корректно перед проведением документа.

{% endnote %}

## Проведение документа

После заполнения всех позиций документ нужно провести. При проведении остатки на складе увеличиваются на указанное количество, а закупочные цены обновляются.

- Проверка корректности данных перед проведением.
- Автоматическое обновление остатков.
- Обновление закупочной цены товаров.

## Редактирование и отмена

Пока документ находится в статусе «Черновик», его можно свободно редактировать — изменять позиции, количество и цены. Черновик можно отменить, если он больше не нужен.

{% note warning %}

После проведения документ становится окончательным — редактирование и отмена невозможны. Если обнаружена ошибка в проведённом приходе, создайте корректирующий документ (списание или возврат поставщику).

{% endnote %}
