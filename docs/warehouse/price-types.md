# Типы цен

Система поддерживает несколько типов цен для каждого товара. Это позволяет гибко управлять ценообразованием — задавать разные цены для розничных продаж, оптовых клиентов и ремонтных услуг.

## Назначение

Типы цен позволяют вести многоуровневое ценообразование. Каждому товару можно присвоить несколько цен, которые используются в разных сценариях — при продаже в розницу, отпуске оптовому клиенту или включении запчасти в стоимость ремонта.

## Предустановленные типы цен

Система содержит базовые типы цен, которые доступны сразу после настройки.

- **Закупочная цена** — цена, по которой товар был приобретён у поставщика (себестоимость). Обновляется автоматически при проведении документа прихода.
- **Ремонтная цена** — цена, по которой запчасть включается в стоимость ремонта. По умолчанию рассчитывается с наценкой 30% от закупочной.
- **Розничная цена** — цена для продажи. По умолчанию рассчитывается с наценкой 50% от закупочной.

## Создание типа цен

Для создания нового типа цен перейдите в **Склад** → **Типы цен**. Укажите название, код и, при необходимости, процент наценки.

- **Название** — отображаемое имя типа цен.
- **Код** — уникальный идентификатор (например, `purchase`, `retail`).
- **Наценка (%)** — процент наценки от закупочной цены для автоматического расчёта.

## Правила наценки

Цена может рассчитываться автоматически на основе наценки от закупочной цены. Это избавляет от необходимости вручную пересчитывать цены при изменении себестоимости.

- Наценка задаётся в процентах от закупочной цены.
- Формула: `цена = закупочная × (1 + наценка / 100)`.
- Если наценка не указана, цена устанавливается вручную.

### Пример расчёта

| Закупочная цена | Наценка | Итоговая цена |
|---|---|---|
| 1 000 руб. | 50% | 1 500,00 руб. |
| 500 руб. | 30% | 650,00 руб. |
| 2 300 руб. | 50% | 3 450,00 руб. |

{% note warning %}

Автоматический пересчёт цен с наценкой происходит только при проведении документа прихода. Если закупочная цена не менялась, остальные цены остаются прежними.

{% endnote %}

## Назначение цен товарам

Цены устанавливаются в карточке товара или массово через импорт. Для каждого товара можно задать значения по каждому типу цен.

- Ручная установка цены в карточке товара.
- Массовое обновление цен через импорт.
- Автоматический расчёт по правилам наценки.

<!-- screenshot: вкладка «Цены» в карточке товара -->

## Использование типов цен

Типы цен применяются в различных сценариях работы системы.

- **Оформление ремонта** — при добавлении запчасти в ремонт подставляется ремонтная цена.
- **Отчёты** — анализ маржинальности на основе закупочной и ремонтной цен.
- **Карточка товара** — отображаются все типы цен для быстрой справки.
