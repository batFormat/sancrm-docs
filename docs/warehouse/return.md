# Возврат поставщику

Документ возврата поставщику фиксирует отправку ранее принятых товаров обратно поставщику. При проведении возврата остатки на складе уменьшаются.

## Когда используется

Возврат оформляется, когда полученные от поставщика товары необходимо вернуть — по причине брака, несоответствия заказу, истечения срока годности или по договорённости с поставщиком.

## Создание документа возврата

Для создания возврата перейдите в раздел складских документов и выберите тип «Возврат поставщику». Укажите склад, поставщика и добавьте позиции.

<!-- screenshot: форма создания документа возврата поставщику -->

### Основные поля

- **Склад** — с какого склада возвращаются товары.
- **Связанный приход** — ссылка на исходный документ прихода (обязательное поле).
- **Поставщик** — контрагент, которому возвращаются товары.
- **Комментарий** — дополнительная информация.

## Связь с документом прихода

Возврат можно связать с конкретным документом прихода. Это позволяет отслеживать, какие именно товары из какой поставки возвращаются, и упрощает работу с поставщиком.

- Выбор документа прихода из списка.
- Автоматическая подстановка поставщика.
- Просмотр позиций оригинального прихода.

{% note info %}

При привязке к документу прихода система подставляет поставщика автоматически и отображает список товаров из выбранной поставки для удобного добавления в возврат.

{% endnote %}

## Причина возврата

Причину возврата можно указать в поле «Комментарий» в свободной форме — например, «Брак», «Несоответствие заказу», «Пересортица» и т. д.

## Добавление товарных позиций

В документ возврата добавляются товары с указанием количества. Если возврат привязан к приходу, доступен быстрый выбор позиций из оригинального документа.

- Выбор позиций из связанного прихода.
- Ручной поиск и добавление товаров.
- Указание количества для возврата.

<!-- screenshot: таблица позиций в документе возврата -->

{% note warning %}

Количество возвращаемых товаров не может превышать текущий остаток на складе. Если остатки недостаточны, проведение документа будет заблокировано.

{% endnote %}

## Проведение документа

При проведении документа возврата остатки на складе уменьшаются. Операция отражается в истории движений товара.

- Уменьшение остатков на складе.
- Фиксация в истории движений.
- Связь с документом прихода в аудите.
