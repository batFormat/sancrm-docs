# Перемещение

Документ перемещения фиксирует передачу товаров с одного склада на другой. При проведении остатки уменьшаются на складе-источнике и увеличиваются на складе-получателе.

## Когда используется

Перемещение применяется, когда необходимо перераспределить товары между складами — например, перевезти запчасти из центрального склада в филиал или переместить товары с основного склада на витрину.

## Создание документа перемещения

Для создания перемещения перейдите в раздел складских документов и выберите тип «Перемещение». Укажите склад-источник, склад-получатель и добавьте позиции.

<!-- screenshot: форма создания документа перемещения -->

### Основные поля

- **Склад-источник** — откуда перемещаются товары.
- **Склад-получатель** — куда перемещаются товары.
- **Комментарий** — дополнительная информация.

{% note warning %}

Склад-источник и склад-получатель должны быть разными. Нельзя создать перемещение внутри одного склада.

{% endnote %}

## Добавление товарных позиций

В документ добавляются товары с указанием количества для перемещения. Доступные остатки проверяются по складу-источнику.

- Поиск товара по наименованию, артикулу или штрихкоду.
- Указание количества для перемещения.
- Контроль остатков на складе-источнике.

<!-- screenshot: таблица позиций в документе перемещения -->

{% note info %}

При добавлении товара в документ отображается текущий остаток на складе-источнике. Это помогает избежать ошибок при указании количества.

{% endnote %}

## Проведение документа

При проведении документа перемещения остатки синхронно обновляются на обоих складах. Операция атомарна — невозможна ситуация, когда товар списан с одного склада, но не зачислен на другой.

- Уменьшение остатков на складе-источнике.
- Увеличение остатков на складе-получателе.
- Фиксация даты и автора перемещения.

## Редактирование и отмена

Пока документ находится в статусе «Черновик», его можно свободно редактировать или отменить. После проведения документ становится окончательным.

{% note warning %}

Проведение перемещения необратимо. Если перемещение выполнено ошибочно, создайте обратное перемещение — с тем же набором позиций, но поменяв местами склад-источник и склад-получатель.

{% endnote %}
