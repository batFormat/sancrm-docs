# Создание первого ремонта

В этом разделе описан полный процесс оформления первого заказа на ремонт — от регистрации клиента до печати квитанции. Пройдя эти шаги, вы освоите основной рабочий сценарий SanCRM.

## Создание клиента

Прежде чем оформить ремонт, необходимо добавить клиента в базу. Здесь будет описано, как создать карточку клиента: указать имя, номер телефона и дополнительные контактные данные. Также будет описан поиск по существующей базе клиентов.

{% note info %}

Если клиент уже есть в базе, система предложит его автоматически при вводе номера телефона или имени.
