# Договор проката

Договор проката — юридический документ, регулирующий отношения между сервисным центром и клиентом при выдаче оборудования во временное пользование. Система автоматически формирует договор на основе настраиваемого шаблона.

<!-- screenshot: пример сформированного договора проката -->

## Автоматическое формирование

Договор проката генерируется автоматически на основе данных заказа. Все поля заполняются из карточки проката без необходимости ручного ввода.

- **Данные клиента** — ФИО, паспортные данные, контактная информация подставляются из карточки клиента.
- **Перечень оборудования** — список выданных позиций с инвентарными номерами и описанием состояния.
- **Условия проката** — сроки, стоимость, сумма залога и порядок расчётов.
- **Ответственность сторон** — условия компенсации при повреждении или утере оборудования.

{% note info %}

Договор формируется в момент сохранения заказа на прокат. При изменении данных заказа (продление срока, добавление позиций) можно сформировать обновлённый договор.

{% endnote %}

## Шаблон договора

Содержание и оформление договора определяются шаблоном, который настраивается администратором.

- **Редактор шаблонов** — визуальный редактор для создания и изменения шаблона договора.
- **Переменные** — в шаблоне используются переменные, которые заменяются данными из заказа (номер договора, ФИО клиента, список позиций, стоимость и т.д.).
- **Логотип и реквизиты** — фирменный стиль, логотип и юридические реквизиты организации.
- **Несколько шаблонов** — возможность создать разные шаблоны для различных типов оборудования или условий проката.

<!-- screenshot: редактор шаблона договора с переменными -->

{% note info %}

Настройка шаблонов договоров доступна в разделе [Настройки](../settings/). Рекомендуется согласовать текст договора с юристом перед началом использования.

{% endnote %}

## Печать договора

Печать доступна из карточки заказа на прокат. Система поддерживает несколько форматов вывода.

- **Печать на принтер** — прямая печать на подключённый принтер.
- **Предварительный просмотр** — просмотр договора перед печатью для проверки корректности.
- **Сохранение в PDF** — экспорт договора в формат PDF для отправки клиенту по email.
- **Печать в двух экземплярах** — по одному экземпляру для клиента и для организации.

<!-- screenshot: окно предварительного просмотра договора перед печатью -->

## Подписание договора

Договор вступает в силу после подписания обеими сторонами. Система фиксирует факт подписания.

- **Место для подписей** — в шаблоне предусмотрены поля для подписи клиента и представителя организации.
- **Дата подписания** — фиксируется автоматически в момент формирования документа.
- **Отметка о подписании** — оператор отмечает в системе, что договор подписан клиентом.

## Процесс возврата

Возврат инструмента — завершающий этап проката, включающий проверку состояния оборудования и окончательный расчёт.

- **Приём инструмента** — оператор принимает оборудование и сверяет комплектацию.
- **Осмотр состояния** — проверка инструмента на наличие повреждений и следов ненадлежащего использования.
- **Фиксация фактической даты возврата** — система рассчитывает итоговую стоимость по фактическому сроку использования.
- **Закрытие заказа** — заказ переводится в статус «Возвращён», позиции инструмента — в статус «Доступен».

<!-- screenshot: форма возврата инструмента с чек-листом осмотра -->

{% note warning %}

При возврате обязательно проверяйте инструмент в присутствии клиента. Если повреждения обнаружены после ухода клиента, взыскание компенсации может быть затруднено.

{% endnote %}

## Обработка повреждений

При обнаружении повреждений или недостачи комплектации система позволяет зафиксировать проблему и рассчитать компенсацию.

- **Фиксация повреждения** — описание выявленного дефекта с возможностью прикрепить фотографии.
- **Оценка ущерба** — расчёт стоимости ремонта или замены на основе балансовой стоимости.
- **Удержание из залога** — автоматический расчёт суммы удержания из залогового платежа.
- **Акт повреждений** — формирование документа, фиксирующего выявленные повреждения и сумму компенсации.

<!-- screenshot: форма фиксации повреждений с полями для описания и фото -->

{% note info %}

Если сумма ущерба превышает размер залога, система сформирует акт с указанием суммы задолженности клиента. Этот документ может быть использован для дальнейшего урегулирования.

{% endnote %}

## Возврат залога

После успешного возврата инструмента залоговый платёж возвращается клиенту. Система автоматизирует этот процесс.

- **Полный возврат** — если инструмент возвращён без повреждений, залог возвращается полностью.
- **Частичный возврат** — при наличии повреждений из залога удерживается сумма компенсации.
- **Квитанция о возврате залога** — формирование документа, подтверждающего возврат залогового платежа.

## Архив договоров

Все сформированные договоры сохраняются в системе и доступны для повторного просмотра и печати.

- **История договоров** — полный архив всех договоров проката с возможностью поиска и фильтрации.
- **Повторная печать** — возможность распечатать любой ранее сформированный договор.
- **Связь с заказом** — каждый договор привязан к конкретному заказу на прокат для быстрого доступа.
