# Фильтры и представления

Фильтры и представления (линзы) позволяют быстро находить нужные ремонты в общем списке и сохранять часто используемые настройки фильтрации для повторного применения.

<!-- screenshot: список ремонтов с панелью фильтров -->

## Панель фильтров

Панель фильтров расположена над списком ремонтов и позволяет отобрать заказы по различным критериям.

- **Статус** — отображение ремонтов в одном или нескольких статусах.
- **Исполнитель** — ремонты конкретного мастера.
- **Клиент** — поиск по имени или телефону клиента.
- **Дата приёма** — ремонты за выбранный период.
- **Срочность** — фильтрация по приоритету.
- **Филиал** — при наличии нескольких филиалов.
- **Тип устройства** — ремонты определённого типа техники.

<!-- screenshot: развёрнутая панель фильтров с заполненными полями -->

## Быстрые фильтры

Помимо детальной фильтрации, доступны быстрые фильтры для наиболее востребованных сценариев.

- **Мои ремонты** — заказы, назначенные текущему пользователю.
- **Просроченные** — ремонты, срок готовности которых истёк.
- **Сегодня** — ремонты, принятые или завершённые сегодня.
- **Без исполнителя** — нераспределённые заказы.

{% note info %}

Быстрые фильтры можно комбинировать с детальными. Например, выбрать «Мои ремонты» и дополнительно отфильтровать по статусу «В работе».

{% endnote %}

## Сохранённые представления (линзы)

Линза — это сохранённый набор фильтров, к которому можно вернуться в любой момент одним кликом. Линзы избавляют от необходимости каждый раз заново настраивать фильтрацию.

### Создание линзы

Настройте нужные фильтры и сохраните комбинацию как новую линзу.

- Задайте понятное название (например, «Срочные в работе» или «Ожидают запчасти»).
- Линза сохраняет все параметры фильтрации, сортировку и видимые колонки.
- Линза может быть личной или общей для всех сотрудников.

<!-- screenshot: диалог сохранения нового представления (линзы) -->

### Управление линзами

Сохранённые линзы доступны в боковой панели или в выпадающем меню над списком ремонтов.

- **Переключение** — быстрое переключение между линзами.
- **Редактирование** — изменение параметров существующей линзы.
- **Удаление** — удаление неактуальных линз.
- **Порядок** — настройка порядка отображения линз.

## Настройка колонок

Список ремонтов отображается в табличном виде, и состав видимых колонок можно настроить.

- Выбор отображаемых колонок из полного списка доступных полей.
- Настройка ширины колонок перетаскиванием.
- Сортировка по любой видимой колонке.
- Конфигурация колонок сохраняется в линзе.

<!-- screenshot: диалог настройки видимых колонок таблицы -->

## Сортировка

Список ремонтов можно отсортировать по любому видимому полю.

- Сортировка по возрастанию или убыванию.
- Многоуровневая сортировка (например, сначала по срочности, затем по дате).
- Параметры сортировки сохраняются в линзе.

## Поиск по списку

Помимо фильтров, доступен полнотекстовый поиск по списку ремонтов.

- Поиск по номеру заказа, имени клиента, описанию неисправности.
- Результаты обновляются по мере ввода.
- Поиск работает совместно с активными фильтрами.

{% note warning %}

Большое количество активных фильтров может замедлить загрузку списка при большом объёме данных. Используйте сохранённые линзы — они работают быстрее, чем повторная ручная настройка фильтров.

{% endnote %}

## Экспорт результатов

Отфильтрованный список ремонтов можно экспортировать для дальнейшего анализа.

- Экспорт в Excel (XLSX).
- Экспорт в CSV.
- Экспортируются только ремонты, соответствующие текущим фильтрам.
- Состав экспортируемых полей соответствует видимым колонкам.
