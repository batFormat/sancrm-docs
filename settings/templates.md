# Шаблоны документов

Шаблоны документов позволяют настроить внешний вид и содержание квитанций, гарантийных талонов, актов приёма-выдачи и других печатных форм. Каждый шаблон можно отредактировать, добавив логотип, юридическую информацию и нужные поля.

<!-- screenshot: список шаблонов документов -->

## Типы шаблонов

Система поддерживает несколько типов документов, каждый из которых имеет собственный шаблон.

- **Квитанция приёма** — документ, подтверждающий приём устройства в ремонт.
- **Гарантийный талон** — документ, подтверждающий гарантию на выполненные работы.
- **Акт выполненных работ** — перечень выполненных работ и использованных запчастей.
- **Наклейка / этикетка** — компактная наклейка для маркировки устройства.
- **Ценник** — шаблон ценника для товаров и услуг.

<!-- screenshot: карточки типов шаблонов -->

## Редактор шаблонов

Встроенный визуальный редактор позволяет изменять структуру и оформление документа без знания HTML или CSS. Доступны инструменты форматирования текста, вставка изображений и таблиц.

- Редактируйте текст непосредственно в области предпросмотра.
- Используйте панель инструментов для форматирования, выравнивания и вставки элементов.
- Переключайтесь между визуальным режимом и режимом исходного кода при необходимости.

## Переменные и плейсхолдеры

Для подстановки динамических данных в шаблон используются переменные. При печати документа переменные автоматически заменяются реальными значениями из карточки ремонта.

### Основные переменные

| Переменная | Описание |
|---|---|
| `{order.number}` | Номер заказа |
| `{order.created_at}` | Дата создания заказа |
| `{order.estimated_date}` | Ориентировочная дата готовности |
| `{client.name}` | ФИО клиента |
| `{client.phone}` | Телефон клиента |
| `{device.type}` | Тип устройства |
| `{device.brand}` | Бренд устройства |
| `{device.model}` | Модель устройства |
| `{device.serial}` | Серийный номер |
| `{order.defect}` | Заявленная неисправность |
| `{order.total}` | Итоговая сумма ремонта |
| `{org.name}` | Название организации |
| `{org.address}` | Адрес организации |
| `{branch.name}` | Название филиала |

{% note info %}

Полный список переменных доступен в боковой панели редактора. Нажмите на переменную, чтобы вставить её в текст шаблона в позиции курсора.

{% endnote %}

## Шаблоны квитанций

Квитанция приёма — основной документ, который выдаётся клиенту при сдаче устройства. Шаблон квитанции обычно включает данные организации, информацию о клиенте, описание устройства и условия обслуживания.

<!-- screenshot: предпросмотр квитанции приёма -->

## Шаблоны гарантийных талонов

Гарантийный талон подтверждает гарантийные обязательства на выполненные работы. Шаблон содержит перечень выполненных работ, срок гарантии и условия её действия.

<!-- screenshot: предпросмотр гарантийного талона -->

## Предпросмотр и печать

Перед сохранением шаблона используйте режим предпросмотра, чтобы увидеть, как документ будет выглядеть с реальными данными. Система подставит тестовые значения во все переменные.

{% note warning %}

После изменения шаблона все новые документы будут формироваться по обновлённой версии. Ранее сгенерированные документы не изменяются.

{% endnote %}
