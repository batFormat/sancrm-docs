# Склады

Настройка складов определяет, где хранятся запчасти и товары вашего сервисного центра. Каждый склад привязывается к филиалу, а остатки отслеживаются раздельно по каждому складу.

<!-- screenshot: список складов в настройках -->

## Создание склада

Для добавления нового склада укажите его название и основные параметры. Название склада используется в складских документах, при списании запчастей на ремонт и в отчётах.

- **Название** — понятное имя склада (например, «Основной склад», «Витрина», «Склад запчастей»).
- **Описание** — дополнительная информация о назначении склада.

<!-- screenshot: форма создания нового склада -->

## Привязка к филиалу

Каждый склад привязывается к одному из филиалов организации. Сотрудники филиала видят остатки на привязанных к нему складах и могут списывать запчасти при выполнении ремонтов.

- К одному филиалу можно привязать несколько складов (например, основной склад и витрину).
- При создании склада выберите филиал из выпадающего списка.

{% note info %}

Если у вас один филиал, все склады автоматически привязываются к нему. Настройка привязки актуальна для организаций с несколькими точками.

{% endnote %}

<!-- screenshot: выбор филиала при настройке склада -->

## Склад по умолчанию

Один из складов можно назначить складом по умолчанию. Он будет автоматически подставляться при создании приходных документов и при списании запчастей на ремонт.

- Склад по умолчанию устанавливается отдельно для каждого филиала.
- При необходимости сотрудник может вручную выбрать другой склад в момент операции.

<!-- screenshot: переключатель склада по умолчанию -->

## Типы складов

В зависимости от назначения склады могут использоваться для разных целей.

- **Основной склад** — хранение запчастей и расходных материалов для ремонтов.
- **Витрина** — товары, доступные для продажи клиентам.
- **Транзитный склад** — временное хранение товаров при перемещении между филиалами.

<!-- screenshot: типы складов в выпадающем списке -->

## Перемещение между складами

Если у вас несколько складов, доступна функция перемещения товаров между ними. Перемещение фиксируется документом и отражается в остатках обоих складов.

{% note warning %}

Удаление склада невозможно, если на нём числятся остатки или к нему привязаны складские документы. Сначала переместите все товары и завершите открытые документы.

{% endnote %}
