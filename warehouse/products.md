# Управление товарами

Каталог товаров — центральный элемент складского модуля. Здесь хранится вся информация о товарах, запчастях и расходных материалах, используемых в работе сервисного центра.

## Каталог товаров

Каталог представляет собой единый список всех товарных позиций с возможностью поиска, фильтрации и сортировки. Каждая позиция содержит основные характеристики, цены и текущие остатки.

<!-- screenshot: общий вид каталога товаров -->

## Карточка товара

Карточка товара содержит подробную информацию о конкретной позиции. Из карточки можно просмотреть историю движений, текущие остатки по складам и настроить цены.

- Наименование и описание.
- Фотографии товара.
- Штрихкод и артикул (SKU).
- Единица измерения.
- Привязка к категории и бренду.

<!-- screenshot: карточка товара с заполненными полями -->

{% note info %}

Штрихкод можно ввести вручную или отсканировать с помощью сканера штрихкодов, подключённого к компьютеру.

{% endnote %}

## Категории товаров

Категории позволяют структурировать каталог и упрощают навигацию. Поддерживается иерархическая структура — категории могут содержать подкатегории.

- Создание и редактирование категорий.
- Вложенность категорий без ограничений по глубине.
- Фильтрация каталога по категории.

<!-- screenshot: дерево категорий товаров -->

## Бренды

Бренды (производители) помогают классифицировать товары по производителю. Поле «Бренд» доступно в карточке товара и может использоваться как фильтр в каталоге.

- Создание и управление списком брендов.
- Привязка товара к бренду.
- Фильтрация по бренду в каталоге.

## Артикул (SKU)

Артикул — уникальный идентификатор товара в вашей системе учёта. Он используется для быстрого поиска и однозначной идентификации позиции.

- Ручной ввод или автоматическая генерация.
- Поиск по артикулу в каталоге и складских документах.
- Уникальность артикула в рамках каталога.

{% note warning %}

Артикул должен быть уникальным. При попытке создать товар с уже существующим артикулом система выдаст предупреждение.

{% endnote %}

## Ценообразование

К каждому товару можно привязать несколько типов цен — розничную, оптовую, закупочную и другие. Подробная настройка ценообразования описана в разделе [Типы цен](price-types.md).

- Закупочная цена (себестоимость).
- Розничная цена.
- Дополнительные типы цен.

## Остатки товара

В карточке товара отображаются текущие остатки по каждому складу. Вы можете быстро оценить наличие и при необходимости перейти к созданию складского документа.

- Остатки в разрезе складов.
- История движений товара.
- Ссылки на связанные складские документы.
