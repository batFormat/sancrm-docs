# Складские документы

Все операции с товарами на складе оформляются через складские документы. Каждый документ фиксирует тип операции, список позиций, дату и ответственного сотрудника.

## Общие сведения

Складские документы являются основой учёта движения товаров. Без проведённого документа остатки на складе не изменяются. Это обеспечивает прозрачность и возможность аудита всех операций.

## Типы документов

Система поддерживает следующие типы складских документов, каждый из которых отвечает за определённый вид операции.

| Тип документа | Назначение | Влияние на остатки |
|---|---|---|
| Приход | Поступление товаров от поставщика | Увеличивает остатки |
| Списание | Удаление товаров с учёта | Уменьшает остатки |
| Перемещение | Передача между складами | Перераспределяет остатки |
| Инвентаризация | Сверка фактических и учётных остатков | Корректирует остатки |
| Возврат поставщику | Возврат ранее принятого товара | Уменьшает остатки |

## Статусы документов

Каждый складской документ проходит через определённые статусы. Остатки изменяются только при проведении документа.

- **Черновик** — документ создан, но не проведён. Остатки не затронуты.
- **Проведён** — документ подтверждён, остатки обновлены.
- **Отменён** — документ отменён, изменения остатков откачены.

{% note warning %}

Отмена проведённого документа возможна только при наличии достаточных остатков. Если товары уже были проданы или списаны, отмена может быть заблокирована.

{% endnote %}

## Создание документа

Процесс создания одинаков для всех типов документов. Вы выбираете тип, указываете склад, добавляете позиции и проводите документ.

1. Перейдите в раздел **Склад** → **Документы**.
2. Нажмите кнопку **Создать документ**.
3. Выберите тип документа.
4. Заполните обязательные поля.
5. Добавьте товарные позиции.
6. Проведите документ.

## Поиск и фильтрация

Список документов можно фильтровать по типу, статусу, дате, складу и ответственному сотруднику. Это позволяет быстро найти нужный документ.

- Фильтр по типу документа.
- Фильтр по периоду.
- Фильтр по складу.
- Полнотекстовый поиск по номеру документа.

## Печать документов

Из любого складского документа можно сформировать печатную форму. Шаблоны печатных форм настраиваются в разделе администрирования.

- Генерация PDF для печати.
- Настраиваемые шаблоны печатных форм.

{% note info %}

Подробнее о настройке шаблонов печати читайте в разделе администрирования.

{% endnote %}
